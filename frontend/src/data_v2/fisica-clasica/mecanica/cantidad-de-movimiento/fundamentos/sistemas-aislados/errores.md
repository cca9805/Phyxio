# Errores frecuentes

## 1- Error de identificación de sistema
Definir el sistema “a medias” y luego aplicar conservación como si fuera aislado.

## 2- Error de interpretación física
Creer que “si hay fuerzas grandes no puede conservarse el momento total”.

## 3- Error de dominio de validez
Aplicar aislamiento cuando el impulso externo no es despreciable (rozamiento fuerte, empuje prolongado, interacción con suelo).

## 4- Error de procedimiento algebraico
No separar por componentes en 2D/3D y mezclar signos o direcciones.

## 5- Error de unidades y parámetros
Usar masas en gramos o velocidades en km/h sin convertir a SI antes de operar.

## 6- Error de cierre de resolución
No comprobar coherencia física: sentido del movimiento, límites (por ejemplo, velocidades absurdas), o conservación por componentes.

## Cómo evitar estos errores de forma sistemática
- Define el sistema por escrito y decide qué queda fuera.
- Justifica el aislamiento con impulso externo (intervalo corto o fuerzas despreciables).
- Trabaja por componentes siempre que haya más de una dirección.
- Verifica unidades SI antes de sustituir.
- Cierra con una interpretación: “¿quién se lleva el movimiento y por qué?”

## Pregunta de control final
Antes de cerrar una resolución de este subtema, pregúntate: **si añado una fuerza externa pequeña pero durante mucho tiempo, ¿seguiría siendo razonable suponer conservación del momento total?**

Si la respuesta es “no”, revisa si tu sistema realmente puede tratarse como aislado.
