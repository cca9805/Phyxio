

# Errores frecuentes

## 1. Error de identificación de variables
Confundir ángulo recorrido con posición angular absoluta, o periodo con frecuencia.

## 2. Error de interpretación física
Olvidar que la aceleración centrípeta apunta siempre hacia el centro y que el signo indica dirección.

## 3. Error de dominio de validez
Aplicar MCU cuando hay aceleración angular o el radio varía.

## 4. Error de unidades y parámetros
Mezclar radianes con grados, segundos con minutos, o no convertir unidades antes de calcular.

## 5. Error de cierre de resolución
Obtener un resultado numérico sin verificar si es físicamente posible (por ejemplo, periodo negativo o velocidad fuera del rango esperado).

## Cómo evitar estos errores de forma sistemática
- Define variables y unidades antes de sustituir.
- Comprueba hipótesis del modelo antes de calcular.
- Cierra siempre con verificación física del resultado y sentido de los signos.
