

# Modelos y validez

## Modelo base del subtema
El Movimiento Circular Uniforme (MCU) describe el desplazamiento de un cuerpo a velocidad angular constante sobre una trayectoria circular. Se asume un sistema idealizado, sin aceleración tangencial, radio constante y sin fuerzas externas que alteren la velocidad.

## Hipótesis operativas
- El móvil se desplaza en una circunferencia de radio fijo.
- La velocidad angular es constante en módulo, dirección y sentido.
- El sistema de referencia, el origen angular y los signos están explícitos.
- Todas las magnitudes se expresan en unidades SI.

## Qué explica bien este modelo
- Permite predecir posición angular, velocidad lineal y aceleración centrípeta en cualquier instante.
- Relaciona periodo, frecuencia, velocidad angular y radio de forma directa y verificable.
- Es la base para el análisis de movimientos circulares más complejos (MCUA, dinámica rotacional).

## Límites del modelo
- No contempla aceleraciones angulares ni variaciones de radio.
- No es válido si existen fuerzas externas que alteran la velocidad angular.
- Si los datos experimentales muestran variaciones sistemáticas, se requiere un modelo más general (MCUA, fuerzas no centrípetas, etc.).

## Señales de que debes cambiar de modelo
- El periodo o la velocidad angular calculados no son constantes en el tiempo.
- La posición predicha no coincide con la observada en más de un punto de control.
- El móvil experimenta aceleración tangencial o el radio varía durante el movimiento.

## Criterio didáctico de cierre
El dominio del MCU implica saber cuándo aplicarlo, justificar sus hipótesis y detectar rápidamente los límites de validez. Un uso correcto del modelo requiere interpretar resultados, verificar unidades y signos, y estar atento a señales que indiquen la necesidad de un marco teórico más completo.
