

# Ejemplos guiados – Movimiento Circular Uniformemente Acelerado (MCUA)

## Ejemplo 1
**Problema:** Un disco parte del reposo ($\omega_0 = 0$) y acelera con $\alpha = 3\ \mathrm{rad/s^2}$ durante $t = 5\ \mathrm{s}$. ¿Cuál es la velocidad angular final y el ángulo girado?

**Solución:**

$\omega = \omega_0 + \alpha t = 0 + 3 \times 5 = 15\ \mathrm{rad/s}$

$\theta = \theta_0 + \omega_0 t + \frac{1}{2}\alpha t^2 = 0 + 0 + \frac{1}{2} \times 3 \times 25 = 37.5\ \mathrm{rad}$

---

## Ejemplo 2
**Cambia un dato:** Si $\alpha = -2\ \mathrm{rad/s^2}$ (frenado), $t = 4\ \mathrm{s}$, $\omega_0 = 8\ \mathrm{rad/s}$.

$\omega = 8 + (-2) \times 4 = 0\ \mathrm{rad/s}$

$\theta = 0 + 8 \times 4 + \frac{1}{2}(-2)\times 16 = 32 - 16 = 16\ \mathrm{rad}$

**Explicación:** El disco se detiene justo al final del intervalo.

---

## Ejemplo 3
- Caso A: Cambia el tiempo a $t=10\ \mathrm{s}$ y repite el cálculo.
- Caso B: Cambia la condición inicial a $\omega_0 = 5\ \mathrm{rad/s}$.
- Caso C: Considera $\alpha = 0$ (verifica que recuperas el caso de MCU).
