

# Modelos y validez – Movimiento Circular Uniformemente Acelerado (MCUA)

## Hipótesis del modelo
- El MCUA describe el movimiento de un cuerpo que gira con aceleración angular constante.
- Se asume que el radio es constante y que no hay deslizamiento.
- Las fuerzas externas no cambian el eje de rotación ni el sentido del movimiento durante el intervalo analizado.
- El sistema es ideal: no hay fricción ni pérdidas de energía.

## ¿Cuándo sí aplica?
- Cuando la aceleración angular es constante y conocida.
- Cuando el radio de giro permanece fijo.
- Cuando las condiciones iniciales (posición angular, velocidad angular) están bien definidas.
- Cuando se puede despreciar la fricción y otros efectos disipativos.

## ¿Cuándo deja de aplicar?
- Si la aceleración angular varía con el tiempo.
- Si el radio cambia (cuerda que se enrolla/desenrolla, etc.).
- Si hay fuerzas externas que alteran el eje de rotación.
- Si existen pérdidas de energía significativas (fricción, rozamiento).

## Cómo ampliar el modelo
- Dividir el movimiento en tramos con aceleraciones distintas.
- Incluir fuerzas externas y analizar la dinámica completa.
- Considerar fricción o resistencia del medio.
- Usar modelos energéticos o ecuaciones diferenciales si la aceleración no es constante.

## Cierre de validez
Reconocer los límites del modelo MCUA permite aplicarlo con criterio y saber cuándo migrar a un marco teórico más general.
