# Ejemplos guiados

## Ejemplo 1- Estado instantáneo
**Enunciado:** Un proyectil es lanzado verticalmente con $v_0=24\ \text{m/s}$. ¿Cuál es la altura máxima y el tiempo total de vuelo?

**Pasos de resolución**
1. Calcula el tiempo de subida: $t_{subida}=v_0/g$.
2. Calcula la altura máxima: $h_{max}=\frac{v_0^2}{2g}$.
3. El tiempo total es $2t_{subida}$.
4. Verifica unidades y sentido físico.

## Ejemplo 2- Variación de velocidad inicial
**Enunciado:** Cambia $v_0$ a $30\ \text{m/s}$ y repite el procedimiento.

**Pasos de resolución**
1. Repite los cálculos y compara la altura máxima y el tiempo total.
2. Explica por qué cambian los resultados.

## Criterio didáctico de cierre
En todos los ejemplos, la clave es justificar hipótesis, mantener consistencia de unidades y verificar que el resultado describe de forma coherente el comportamiento balístico.
