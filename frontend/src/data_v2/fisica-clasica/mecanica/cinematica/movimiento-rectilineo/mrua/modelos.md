
# Modelos y validez

## Modelo base del subtema
El Movimiento Rectilíneo Uniformemente Acelerado (MRUA) describe el desplazamiento de un cuerpo sometido a aceleración constante en línea recta. Se asume un sistema idealizado, sin variaciones de aceleración ni fuerzas externas cambiantes, y parámetros constantes durante el intervalo de análisis.

## Hipótesis operativas
- El móvil se desplaza en una sola dimensión, con aceleración constante.
- La velocidad inicial y la posición inicial están bien definidas.
- El sistema de referencia, el origen temporal y espacial, y los signos están explícitos.
- Todas las magnitudes se expresan en unidades SI.

## Qué explica bien este modelo
- Permite predecir posición y velocidad en cualquier instante a partir de condiciones iniciales.
- Relaciona desplazamiento, velocidad, aceleración y tiempo de forma directa y verificable.
- Es la base para el análisis de trayectorias con tramos de aceleración constante.

## Límites del modelo
- No contempla aceleraciones variables ni fuerzas externas no constantes.
- No es válido si existen rozamientos variables, pendientes cambiantes o fuerzas no compensadas.
- Si los datos experimentales muestran variaciones sistemáticas, se requiere un modelo más general (aceleración variable, dinámica avanzada, etc.).

## Señales de que debes cambiar de modelo
- El tiempo calculado resulta negativo o físicamente imposible.
- La posición predicha no coincide con la observada en más de un punto de control.
- El móvil cambia de aceleración durante el intervalo analizado.

## Criterio didáctico de cierre
El dominio del MRUA implica saber cuándo aplicarlo, justificar sus hipótesis y detectar rápidamente los límites de validez. Un uso correcto del modelo requiere interpretar resultados, verificar unidades y signos, y estar atento a señales que indiquen la necesidad de un marco teórico más completo.
