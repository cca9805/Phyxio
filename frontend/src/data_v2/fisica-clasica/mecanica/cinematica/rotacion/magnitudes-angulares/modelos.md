
# Modelos y validez

## Modelo base del subtema
Las magnitudes angulares (posición angular, velocidad angular, aceleración angular) permiten describir el movimiento rotacional de un cuerpo respecto a un eje fijo. El modelo asume un sistema idealizado, con ejes y signos definidos, y sin variaciones bruscas de las condiciones durante el intervalo de análisis.

## Hipótesis operativas
- El eje de rotación es fijo y conocido.
- Las magnitudes se definen respecto a un sistema de referencia explícito.
- Las unidades empleadas son SI (radianes, rad/s, rad/s²).
- No hay ambigüedad en el sentido de giro ni en la elección del origen angular.

## Qué explica bien este modelo
- Permite calcular desplazamientos angulares, velocidades y aceleraciones a partir de datos experimentales o condiciones iniciales.
- Relaciona magnitudes angulares con magnitudes lineales (por ejemplo, $v = \omega R$).
- Es la base para el análisis de sistemas rotacionales más complejos (MCU, MCUA, dinámica rotacional).

## Límites del modelo
- No contempla variaciones del eje de rotación ni movimientos fuera del plano.
- No es válido si existen discontinuidades o saltos bruscos en las condiciones iniciales.
- Si los datos experimentales muestran inconsistencias, se requiere un modelo más general.

## Señales de que debes cambiar de modelo
- El desplazamiento angular calculado no coincide con la observación.
- Las unidades o el sentido de giro no son coherentes en los resultados.
- El sistema presenta aceleraciones no previstas o el eje de rotación cambia.

## Criterio didáctico de cierre
Dominar las magnitudes angulares implica saber cuándo aplicarlas, justificar sus hipótesis y detectar rápidamente los límites de validez. Un uso correcto requiere interpretar resultados, verificar unidades y signos, y estar atento a señales que indiquen la necesidad de un marco teórico más completo.
