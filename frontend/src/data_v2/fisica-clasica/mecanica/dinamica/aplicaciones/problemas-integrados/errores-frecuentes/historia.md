# Errores Frecuentes - Historia

El analisis sistematico de errores se incorporo a la didactica de la fisica para pasar de una evaluacion de resultado a una evaluacion de razonamiento.

En mecanica, esta evolucion fue clave: dos alumnos pueden obtener numeros parecidos, pero solo uno puede justificar el modelo correcto.

Este subtema continua esa linea: priorizar metodo verificable frente a resoluciones opacas.
