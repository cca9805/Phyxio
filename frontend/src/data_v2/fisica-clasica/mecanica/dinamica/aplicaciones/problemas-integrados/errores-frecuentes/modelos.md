# Errores Frecuentes - Modelos

## Modelo correcto de referencia
Se usa una ecuacion dinamica validada por DCL y proyecciones consistentes.

## Modelo con error de seno/coseno
Permite cuantificar cuanto cambia la aceleracion por un fallo trigonometrico comun.

## Modelo sin rozamiento
Se usa como caso de control para comparar sesgo cuando se omite \(\mu\).

## Modelo de error cuantificado
Calcula error absoluto y relativo para transformar la correccion en evidencia numerica.
