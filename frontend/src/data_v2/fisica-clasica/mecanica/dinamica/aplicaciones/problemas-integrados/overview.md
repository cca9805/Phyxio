## Idea general del tema
En este tema se entrena la resolucion completa de problemas de dinamica donde intervienen varias ideas a la vez: fuerzas, rozamiento, ejes, segunda ley de Newton y, cuando procede, cinematica.
No es un bloque de formulas sueltas. Es un bloque de criterio fisico: decidir que modelo usar, justificar cada paso y validar el resultado.

## Que vas a comprender de verdad
- Traducir enunciados largos a un esquema fisico claro.
- Separar datos utiles de datos distractores.
- Construir un DCL robusto y ecuaciones por ejes.
- Enlazar dinamica y cinematica sin mezclar supuestos.
- Detectar errores tipicos antes de entregar una solucion.

- Problemas tipo examen.
- Errores frecuentes.

- Que significa "plantear bien" un problema de dinamica
- Como saber si una ecuacion aplica o no aplica en ese caso
- Como comprobar que el resultado numerico tiene sentido fisico
- Como reducir fallos bajo presion de tiempo

## Ruta didactica recomendada
1. Empieza por `problemas-tipo-examen` para fijar el metodo completo.
2. Pasa a `errores-frecuentes` para blindar el procedimiento.
3. Repite con variaciones de datos y compara sensibilidad del resultado.

## Criterio fisico que debes adquirir
Debes identificar cuando el modelo simplificado es valido y cuando requiere correcciones.

## Conexion con problemas reales
Relaciona este tema con aplicaciones reales y situaciones experimentales.

## Objetivo final del tema
Al terminar, debes poder resolver problemas integrados de dinamica con trazabilidad completa: modelo, ecuaciones, calculo, chequeo e interpretacion.
