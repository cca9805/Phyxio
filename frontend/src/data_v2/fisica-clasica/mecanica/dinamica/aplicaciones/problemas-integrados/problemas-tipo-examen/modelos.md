# Problemas Tipo Examen - Modelos

## Modelo 1- Bloque en plano inclinado
Ecuaciones base: descomposicion de peso, normal, rozamiento y segunda ley en el eje del movimiento.

## Modelo 2- Fuerza externa inclinada
La fuerza aplicada modifica simultaneamente componente tangencial y normal, por lo que cambia tambien el rozamiento.

## Modelo 3- Sistema por fases
Primera fase dinamica para hallar \(a\), segunda fase cinematica para \(v\) y \(x\).

## Modelo 4- Analisis de sensibilidad
Se estudia como cambian \(a\), \(v\) y \(x\) al variar \(\mu\), \(\theta\), \(F\) o \(m\).
