# Analogia Traslacional · Modelos

Modelos base implicados:

## 1- Modelo lineal

$$
F=ma
$$

Describe respuesta traslacional de una masa bajo fuerza neta.

## 2- Modelo angular

$$
\tau=I\alpha
$$

Describe respuesta rotacional de un cuerpo bajo torque neto.

## 3- Modelo de acople (cuando hay rodadura/circular)

$$
v=\omega r,\quad a_t=\alpha r
$$

Permite traducir resultados entre variables lineales y angulares.

Criterio de uso:
- la analogia guia el razonamiento,
- el modelo de acople cierra la traduccion cuantitativa.
