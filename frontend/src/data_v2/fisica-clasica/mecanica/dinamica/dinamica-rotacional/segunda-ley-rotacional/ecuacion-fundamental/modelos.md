# Ecuacion Fundamental · Modelos

Modelos que convergen en este subtema:

## 1- Modelo dinamico rotacional

$$
\sum\tau = I\alpha
$$

Modelo principal para predecir cambio de velocidad angular.

## 2- Modelo de torque neto

$$
\sum\tau = \tau_1+\tau_2+\cdots
$$

Define la causa efectiva del giro tras sumar contribuciones con signo.

## 3- Modelo limite de equilibrio

$$
\sum\tau=0 \Rightarrow \alpha=0
$$

Caso limite para equilibrio rotacional o giro uniforme.

Criterio practico:
- usa el modelo principal,
- apoyate en los modelos auxiliares para interpretar casos especiales.
