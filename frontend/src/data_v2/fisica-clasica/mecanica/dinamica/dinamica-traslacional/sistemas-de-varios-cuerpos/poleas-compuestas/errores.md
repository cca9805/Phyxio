# Errores comunes

- Contar poleas en vez de contar tramos activos de soporte.
- Asumir que reducir fuerza implica reducir trabajo total.
- Olvidar corregir por rendimiento cuando el problema no es ideal.
- Perder coherencia entre fuerza calculada y recorrido necesario.
- No comprobar si el resultado es viable para el operador o actuador.
