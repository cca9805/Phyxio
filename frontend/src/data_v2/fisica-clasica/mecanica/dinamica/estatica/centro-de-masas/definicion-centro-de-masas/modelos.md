# Definicion Centro De Masas - Modelo y validez

## Idea clave
El modelo base representa la distribucion de masa mediante un punto equivalente para describir posicion global y analisis mecanico.

## Hipotesis de trabajo
- Regimen clasico no relativista.
- Coordenadas medidas en un sistema inercial.
- Masas positivas y bien definidas.

## Modelo usado en este subtema
- Sistema discreto elemental de dos masas en 1D para fijar concepto.
- Extension conceptual a N masas y distribucion continua.

## Donde deja de ser suficiente
- Sistemas deformables con redistribucion de masa dependiente del tiempo.
- Escalas relativistas.
- Analisis donde importan tensiones internas, no solo posicion global.
