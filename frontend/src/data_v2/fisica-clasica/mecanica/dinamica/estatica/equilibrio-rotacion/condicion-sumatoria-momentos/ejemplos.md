# Ejemplos

## 1- Dos momentos opuestos
Si \(M_1=120\,\mathrm{N\,m}\), para equilibrio:

$$
M_1 + M_2 = 0 \Rightarrow M_2 = -120\,\mathrm{N\,m}
$$

## 2- Viga con una carga
Datos: \(R=500\,\mathrm{N}\), \(d_R=0.8\,\mathrm{m}\), \(d_P=2.0\,\mathrm{m}\).

$$
P = \frac{R d_R}{d_P} = \frac{500\cdot 0.8}{2.0} = 200\,\mathrm{N}
$$

## 3- Brazo necesario
Datos: \(P=150\,\mathrm{N}\), \(R=300\,\mathrm{N}\), \(d_R=0.6\,\mathrm{m}\).

$$
d_P = \frac{R d_R}{P} = \frac{300\cdot 0.6}{150} = 1.2\,\mathrm{m}
$$
