# Modelos y validez

## 1- Cuerpo rigido ideal
Se desprecia deformacion de la barra o pieza para aislar el efecto de giro.

## 2- Punto/eje de referencia fijo
El momento siempre se define respecto a un eje concreto; cambiar eje cambia cada momento individual.

## 3- Regimen plano
En este subtema se trabaja principalmente en 2D, con un unico eje perpendicular al plano.
