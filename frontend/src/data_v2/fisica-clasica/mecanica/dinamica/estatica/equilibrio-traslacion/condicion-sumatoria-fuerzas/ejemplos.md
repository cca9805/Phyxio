# Ejemplos

## 1) Tres fuerzas concurrentes
Dos fuerzas conocidas \(\vec{F}_1\) y \(\vec{F}_2\) actúan sobre un punto. Encuentra \(\vec{F}_3\) para equilibrio:

\[
\vec{F}_3 = -(\vec{F}_1+\vec{F}_2)
\]

Luego puedes obtener módulo y ángulo con trigonometría.

---

## 2) Lámpara colgada (idea)
En un nudo con dos cuerdas, la suma de tensiones y el peso deben cumplir:

\[
\sum F_x = 0,\quad \sum F_y=0
\]

La clave es proyectar bien las tensiones por el ángulo de cada cuerda.
