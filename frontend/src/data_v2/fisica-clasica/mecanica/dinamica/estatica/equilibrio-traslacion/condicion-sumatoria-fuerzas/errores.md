# Errores comunes

1) Confundir “equilibrio” con “no se mueve”: también hay equilibrio con velocidad constante.
2) Olvidar que ΣF = 0 es una condición vectorial: hay que cumplir ΣFx = 0 y ΣFy = 0 a la vez.
3) Signos mal puestos por elegir ejes sin criterio.
4) Proyectar mal fuerzas inclinadas (sin trigonometría correcta).
5) Dibujar fuerzas que no existen (por ejemplo “centrípeta” como agente) en un DCL.
