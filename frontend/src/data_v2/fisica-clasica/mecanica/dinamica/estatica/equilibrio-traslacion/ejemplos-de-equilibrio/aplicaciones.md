# Aplicaciones

- Diseno de apoyos y estructuras simples para que no deslicen ni colapsen.
- Seleccion de materiales de contacto con coeficiente de rozamiento adecuado.
- Analisis de tensiones en cuerdas, cables y sistemas suspendidos.
- Evaluacion de seguridad en transporte y estiba de cargas.
- Base para pasar de estatica traslacional a equilibrio con momentos.
