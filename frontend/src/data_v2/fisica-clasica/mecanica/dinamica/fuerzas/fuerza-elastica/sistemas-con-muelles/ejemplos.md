
# Ejemplos guiados

## Ejemplo 1 – Muelles en paralelo
**Enunciado:** Dos muelles de $k_1 = 200\,\mathrm{N/m}$ y $k_2 = 300\,\mathrm{N/m}$ se conectan en paralelo. ¿Cuál es la constante equivalente?

**Pasos de resolución**
1. Identifica la configuración: paralelo (ambos extremos comunes).
2. Aplica la fórmula: $k_{eq} = k_1 + k_2$.
3. Sustituye: $k_{eq} = 200 + 300 = 500\,\mathrm{N/m}$.
4. Interpreta: el sistema es más rígido que cada muelle por separado.

## Ejemplo 2 – Muelles en serie
**Enunciado:** Dos muelles de $k_1 = 200\,\mathrm{N/m}$ y $k_2 = 300\,\mathrm{N/m}$ se conectan en serie. ¿Cuál es la constante equivalente?

**Pasos de resolución**
1. Identifica la configuración: serie (uno tras otro).
2. Aplica la fórmula: $\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2}$.
3. Sustituye: $k_{eq} = \frac{200 \times 300}{200 + 300} = 120\,\mathrm{N/m}$.
4. Interpreta: el sistema es menos rígido que cada muelle por separado.

## Criterio didáctico de cierre
En estos ejemplos, es fundamental justificar la elección de la fórmula según la configuración, distinguir entre serie y paralelo, y verificar que la interpretación física corresponde al comportamiento real del sistema.
