
# Errores frecuentes

## 1- Error de suma de constantes
Sumar constantes en serie (en serie se suman inversas: $\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2}$).

## 2- Error de suma de inversas
Sumar inversas en paralelo (en paralelo se suman directamente: $k_{eq} = k_1 + k_2$).

## 3- Error de identificación de variable común
Confundir qué es común: en serie la fuerza es común, en paralelo la elongación es común.

## 4- Error de unidades
Mezclar unidades (N/m vs N/cm) sin convertir.

## 5- Error de reparto de elongaciones
Usar la elongación total como la de un solo muelle sin justificar el reparto según la configuración.

## 6- Error de cierre de resolución
No verificar si la constante equivalente y las elongaciones calculadas son coherentes con la configuración y las condiciones del sistema.

## Cómo evitar estos errores de forma sistemática
- Dibuja siempre el esquema del sistema y define la configuración.
- Aplica las fórmulas correctas según serie o paralelo.
- Cierra cada ejercicio validando la coherencia física y matemática del resultado.
