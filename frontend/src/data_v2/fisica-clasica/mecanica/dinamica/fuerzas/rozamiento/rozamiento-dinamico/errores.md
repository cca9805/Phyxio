
# Errores frecuentes

## 1- Error de identificación de coeficiente
Usar $\mu_s$ cuando el cuerpo ya desliza (debe ser $\mu_k$).

## 2- Error de dirección
Dibujar el rozamiento en el mismo sentido del movimiento. El rozamiento siempre se opone al deslizamiento relativo.

## 3- Error de dependencia
Olvidar que el rozamiento dinámico depende de la normal $N$.

## 4- Confusión conceptual
Confundir “se opone al movimiento” con “se opone al deslizamiento relativo”. El rozamiento actúa contra el movimiento relativo de las superficies.

## 5- Error de modelo
Suponer que $f_k$ cambia con la velocidad en el modelo básico de Coulomb. En este modelo, $f_k$ es constante mientras dure el deslizamiento.

## 6- Error de cierre de resolución
No verificar si el sistema está realmente en régimen dinámico y si el valor calculado de $f_k$ es coherente con la situación física.

## Cómo evitar estos errores de forma sistemática
- Distingue claramente entre rozamiento estático y dinámico en cada problema.
- Usa el coeficiente correcto según el régimen.
- Cierra cada ejercicio validando el sentido, magnitud y régimen del rozamiento involucrado.
