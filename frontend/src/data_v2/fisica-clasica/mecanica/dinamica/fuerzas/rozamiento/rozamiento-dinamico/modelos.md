
# Modelos y validez

## Modelo base del subtema
El rozamiento dinámico (cinético) se modela como la fuerza que se opone al movimiento relativo entre dos superficies en contacto seco, una vez iniciado el deslizamiento. Se utiliza el modelo clásico de Coulomb, con coeficiente de rozamiento dinámico $\mu_k$ constante.

## Hipótesis operativas
- Deslizamiento ya iniciado entre las superficies.
- Superficies macroscópicas, rígidas y secas.
- El coeficiente $\mu_k$ es constante para el par de materiales y condiciones dadas.
- Se desprecia la dependencia con la velocidad en el modelo básico.
- La normal $N$ está bien definida y calculada según la geometría.

## Qué explica bien este modelo
- La fuerza de rozamiento durante el deslizamiento: $f_k = \mu_k N$.
- La independencia de $f_k$ respecto a la velocidad (en el modelo clásico).
- La diferencia entre rozamiento estático y dinámico.

## Límites del modelo
- Si hay lubricación, el modelo pierde validez.
- Si el coeficiente varía con la velocidad, temperatura o desgaste, se requieren modelos más avanzados.
- Si las superficies se deforman apreciablemente, el modelo clásico es solo una aproximación.

## Señales de que debes cambiar de modelo
- Se observa que $f_k$ depende de la velocidad o de otras variables externas.
- Hay efectos de adherencia, desgaste o generación de calor significativos.
- El rozamiento no es proporcional a la normal en el experimento.
