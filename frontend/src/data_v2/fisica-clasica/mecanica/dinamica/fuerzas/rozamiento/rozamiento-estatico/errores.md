
# Errores frecuentes

## 1- Error de identificación de régimen
Suponer siempre $f = \mu_s N$. En realidad, $f$ se ajusta hasta $f_{s,\max}$ y solo alcanza ese valor justo antes de deslizar.

## 2- Error de existencia
Creer que el rozamiento solo existe cuando hay movimiento. El rozamiento estático actúa mientras no hay deslizamiento.

## 3- Error de dirección
Dibujar el rozamiento en el mismo sentido que la fuerza aplicada. El rozamiento siempre se opone al posible movimiento relativo.

## 4- Error de ajuste
Olvidar que el rozamiento estático se ajusta exactamente al valor necesario para mantener el equilibrio, hasta su máximo.

## 5- Confusión de regímenes
Confundir rozamiento estático con dinámico: el primero impide el inicio del movimiento, el segundo actúa durante el deslizamiento.

## 6- Error de cierre de resolución
No verificar si la fuerza aplicada supera $f_{s,\max}$ y, por tanto, si el sistema permanece en reposo o comienza a deslizar.

## Cómo evitar estos errores de forma sistemática
- Distingue claramente entre rozamiento estático y dinámico en cada problema.
- Comprueba si la fuerza aplicada supera el umbral antes de suponer movimiento.
- Cierra cada ejercicio validando el sentido, magnitud y régimen del rozamiento involucrado.
