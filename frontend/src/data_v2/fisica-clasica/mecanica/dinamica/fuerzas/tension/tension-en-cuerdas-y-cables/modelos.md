# Modelo y validez (Tensión)

## Suposiciones del modelo ideal
- Cuerda/cable de masa despreciable (λ≈0).
- Inextensible (no se alarga de forma apreciable).
- Poleas ideales: sin rozamiento y sin masa.
- La tensión es la misma en todos los puntos de una misma cuerda ideal.

## Dominio de validez
El modelo ideal funciona bien cuando:
- la cuerda es ligera comparada con las masas del sistema,
- las poleas tienen rozamiento despreciable,
- las deformaciones del cable no afectan al resultado.

## Cuándo NO aplicar el modelo ideal
- Cables largos y pesados (λ no despreciable): la tensión varía con la posición.
- Poleas con rozamiento o inercia: la tensión puede diferir entre tramos.
- Cables elásticos: aparece deformación apreciable (ingeniería).

## Casos límite útiles
- a = 0 → equilibrio o velocidad constante → en vertical T = mg.
- m1 = m2 (Atwood) → a = 0 → T = m1 g = m2 g.
- g → 0 (microgravedad) → el concepto de tensión sigue existiendo, pero el peso deja de dominar.
