# Errores frecuentes

## 1- Error de identificación de fuerzas
Dibujar fuerzas de acción–reacción en el mismo DCL o incluir fuerzas inexistentes.

## 2- Error de omisión
Omitir fuerzas reales como rozamiento o normal.

## 3- Error de referencia
Elegir ejes que complican innecesariamente el análisis.

## 4- Error de sistema
Mezclar varios cuerpos sin justificar el sistema compuesto.

## 5- Error de cierre de resolución
No verificar que el DCL representa todas las fuerzas relevantes y que el sistema de referencia es coherente.

## Cómo evitar estos errores de forma sistemática
- Define todas las fuerzas antes de dibujar.
- Justifica el sistema compuesto y los ejes elegidos.
- Cierra siempre con verificación física del resultado.
