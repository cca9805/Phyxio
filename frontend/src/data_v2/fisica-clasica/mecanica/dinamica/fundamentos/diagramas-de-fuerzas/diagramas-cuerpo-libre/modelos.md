# Modelos y validez

## Modelo base del subtema
El Diagrama de Cuerpo Libre (DCL) es la herramienta fundamental para analizar fuerzas y movimientos en dinámica clásica. Permite representar todas las fuerzas reales que actúan sobre un cuerpo, idealizado como puntual o rígido, en un sistema inercial.

## Hipótesis operativas
- Solo se dibujan fuerzas reales (peso, normal, tensión, rozamiento, etc.).
- El cuerpo se considera aislado, sin interacción con otros salvo que se justifique sistema compuesto.
- El sistema de referencia y los ejes deben definirse explícitamente.

## Qué explica bien este modelo
- Permite identificar y sumar todas las fuerzas para aplicar las leyes de Newton.
- Es la base para resolver problemas de dinámica, estática y vibraciones.

## Límites del modelo
- No incluye fuerzas ficticias salvo que se trabaje en sistemas no inerciales.
- No representa fuerzas internas en sistemas compuestos, salvo análisis específico.

## Señales de que debes cambiar de modelo
- Si el sistema no es inercial, se requieren fuerzas ficticias.
- Si hay interacción entre varios cuerpos, se debe justificar el sistema compuesto.

## Cierre didáctico
Dominar el DCL es esencial para abordar cualquier problema de dinámica con rigor y trazabilidad.
