# Modelos y alcance

- Fuerzas reales clásicas (contacto, cuerda, muelle, gravedad).
- Rozamiento con modelo simple: cinético \(f_k=\mu N\) y estático \(f_s \le \mu_s N\).
- No se entra aquí en fuerzas ficticias (solo sistemas no inerciales).
- Los valores numéricos de N, T, etc. se calculan con Newton en hojas de aplicación.
