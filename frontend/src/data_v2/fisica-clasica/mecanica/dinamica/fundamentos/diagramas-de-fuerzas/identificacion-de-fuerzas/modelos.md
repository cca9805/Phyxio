# Modelos y validez

## Modelo base del subtema
Este subtema modela la identificación y representación de todas las fuerzas reales que actúan sobre un cuerpo aislado, usando el diagrama de cuerpo libre (DCL) como herramienta central. Se consideran fuerzas de contacto (normal, rozamiento, tensión, elástica) y fuerzas a distancia (peso, empuje).

## Hipótesis operativas
- El cuerpo está claramente aislado del entorno para el análisis.
- Solo se representan fuerzas reales, nunca pares acción–reacción en el mismo DCL.
- El rozamiento se modela con leyes simples: $f_k=\mu N$ (cinético), $f_s \leq \mu_s N$ (estático).

## Qué explica bien este modelo
- Permite inventariar todas las fuerzas que actúan sobre un cuerpo.
- Facilita la aplicación correcta de la segunda ley de Newton.
- Ayuda a evitar errores de confusión entre fuerzas reales y ficticias.

## Límites del modelo
- No incluye fuerzas ficticias (solo para marcos no inerciales).
- El modelo de rozamiento es una aproximación válida solo en casos simples.
- No representa pares acción–reacción en el mismo DCL.

## Señales de que debes cambiar de modelo
- Si aparecen fuerzas que no pueden justificarse físicamente.
- Si el DCL incluye fuerzas de otros cuerpos o pares acción–reacción.
- Si el modelo de rozamiento no describe bien el fenómeno observado.
