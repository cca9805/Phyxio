# Ejemplos

## 1) Aceleración por fuerza
Un bloque de masa \(m=2\,\text{kg}\) tiene fuerza neta \(F_x=6\,\text{N}\).

\[
a_x = \frac{F_x}{m} = \frac{6}{2} = 3\,\text{m/s}^2
\]

## 2) Fuerza necesaria
Quieres \(a_x=1.5\,\text{m/s}^2\) para \(m=4\,\text{kg}\).

\[
F_x = m a_x = 4\cdot 1.5 = 6\,\text{N}
\]

## 3) Caso con fuerza neta negativa
Si \(F_x=-10\,\text{N}\) y \(m=5\,\text{kg}\):

\[
a_x = \frac{-10}{5} = -2\,\text{m/s}^2
\]
