# Aplicaciones

## Aplicaciones científicas y tecnológicas
- Elección correcta del sistema de referencia en problemas de dinámica y estática.
- Análisis de movimientos aparentes y base para introducir fuerzas ficticias en marcos acelerados.
- Ingeniería, navegación, simulaciones físicas y diseño de experimentos de precisión.

## Qué se calcula en estas aplicaciones
- Movimientos reales y aparentes, fuerzas netas y ficticias.
- Validación de límites operativos y coherencia experimental.
- Compatibilidad de resultados entre diferentes sistemas de referencia.

## Decisiones de diseño y operación basadas en este subtema
- Definir correctamente el sistema de referencia y justificar su validez antes de aplicar las leyes de Newton.
- Comprobar coherencia entre magnitudes medidas y transformadas.
- Justificar hipótesis y condiciones de validez antes de aplicar el modelo inercial.

## Lectura didáctica
El objetivo no es solo obtener un número: es decidir si el modelo elegido describe bien el sistema físico y permite tomar decisiones técnicas con criterio. La clave es identificar cuándo el sistema de referencia es realmente inercial y cuándo se requieren correcciones o modelos más generales.
