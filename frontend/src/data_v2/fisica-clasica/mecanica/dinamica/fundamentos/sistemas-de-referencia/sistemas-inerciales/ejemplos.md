# Ejemplos guiados

## Ejemplo 1- Tren a velocidad constante
**Enunciado:** Dentro de un tren que se mueve a velocidad constante, una pelota lanzada verticalmente cae en el mismo punto. ¿Por qué el tren es un sistema inercial?

**Pasos de resolución**
1. Identifica el sistema de referencia: tren en MRU respecto a la Tierra.
2. No hay aceleración del sistema: no se requieren fuerzas ficticias.
3. Se aplican las leyes de Newton sin correcciones.
4. La pelota sigue una trayectoria vertical en el tren y en la Tierra.

## Ejemplo 2- Coche acelerando
**Enunciado:** Un coche acelera y una pelota lanzada verticalmente se desplaza hacia atrás. ¿Por qué el coche no es un sistema inercial?

**Pasos de resolución**
1. El sistema de referencia (coche) está acelerado respecto a la Tierra.
2. Se requieren fuerzas ficticias para explicar el movimiento desde el coche.
3. Las leyes de Newton no se aplican directamente sin correcciones.

## Ejemplo 3- Nave en el espacio
**Enunciado:** Una nave en el espacio, lejos de fuerzas externas, se considera un sistema prácticamente inercial. ¿Por qué?

**Pasos de resolución**
1. No hay fuerzas externas apreciables: la nave no acelera.
2. El sistema de referencia de la nave es inercial para todos los efectos prácticos.
3. Se aplican las leyes de Newton sin necesidad de fuerzas ficticias.

## Criterio didáctico de cierre
En todos los ejemplos, la clave es justificar hipótesis, mantener consistencia de unidades y verificar que el resultado describe de forma coherente el comportamiento físico esperado en cada sistema de referencia.
