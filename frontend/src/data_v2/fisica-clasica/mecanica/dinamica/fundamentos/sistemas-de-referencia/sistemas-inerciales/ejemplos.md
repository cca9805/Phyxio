# Ejemplos

## 1) Tren a velocidad constante
Dentro del tren, una pelota lanzada verticalmente cae en el mismo punto.
El tren es un sistema inercial.

## 2) Coche acelerando
La pelota se desplaza hacia atrás.
El coche no es un sistema inercial.

## 3) Nave en el espacio
Si no actúan fuerzas externas, es un sistema prácticamente inercial.
