# Errores frecuentes

## 1- Error de identificación de sistema
Creer que cualquier sistema sirve para aplicar Newton, sin verificar si es inercial.

## 2- Error de omisión
Pensar que la Tierra es siempre exactamente inercial, incluso en experimentos de alta precisión o en marcos acelerados.

## 3- Error de interpretación física
Confundir velocidad constante con aceleración cero del sistema, o no distinguir entre marcos acelerados y no acelerados.

## 4- Error de cierre de resolución
Aplicar $\sum F = ma$ en un sistema acelerado sin añadir fuerzas ficticias, o no justificar la validez del marco elegido.

## 5- Error de unidades y parámetros
No definir correctamente los ejes, signos o referencias temporales en el sistema de referencia.

## Cómo evitar estos errores de forma sistemática
- Define correctamente el sistema de referencia antes de aplicar las leyes de Newton.
- Comprueba hipótesis y coherencia física antes de calcular.
- Justifica la validez del marco elegido y revisa si se requieren correcciones por aceleración.
- Cierra siempre con verificación física del resultado y compatibilidad con la hipótesis inercial.
