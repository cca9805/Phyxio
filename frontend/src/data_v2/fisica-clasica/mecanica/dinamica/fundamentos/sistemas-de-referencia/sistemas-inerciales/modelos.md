# Modelos y validez

## Modelo base del subtema
Este subtema modela el sistema de referencia en el que las leyes de Newton se aplican sin necesidad de introducir fuerzas ficticias. Se asume un marco clásico, determinista y sin aceleraciones ocultas, con parámetros constantes durante el análisis.

## Hipótesis operativas
- El sistema de referencia está en reposo o movimiento rectilíneo uniforme respecto a un marco absoluto (estrellas fijas).
- La Tierra se considera inercial para la mayoría de problemas prácticos, salvo estudios de alta precisión.
- Unidades SI y definición explícita de ejes, signos y referencia temporal.

## Qué explica bien este modelo
- Permite aplicar directamente las leyes de Newton y analizar movimientos reales.
- Es la base para distinguir entre fuerzas reales y ficticias.
- Permite validar la coherencia de experimentos y simulaciones físicas.

## Límites del modelo
- Si el sistema está acelerado, se requieren fuerzas ficticias y un modelo más general.
- En estudios de precisión, la Tierra no es perfectamente inercial y se deben considerar correcciones.

## Señales de que debes cambiar de modelo
- Desviaciones sistemáticas entre medida y predicción en experimentos o simulaciones.
- Dependencia clara de resultados con la aceleración del sistema de referencia.
- Incoherencias físicas persistentes aun con unidades y signos correctos.

## Cierre didáctico
El dominio real del subtema está en saber cuándo y cómo aplicar el modelo inercial, y cuándo migrar a un modelo más completo. La clave es identificar el marco adecuado antes de aplicar las leyes de Newton y justificar la validez de las hipótesis en cada situación.
