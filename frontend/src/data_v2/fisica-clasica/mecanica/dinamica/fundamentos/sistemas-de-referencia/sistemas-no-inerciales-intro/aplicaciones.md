# Aplicaciones

## Aplicaciones científicas y tecnológicas
- Ascensores y “peso aparente”.
- Vehículos que aceleran o frenan (coches, trenes, aviones).
- Trenes en curvas y sistemas rotantes (a nivel conceptual).
- Meteorología y navegación (Coriolis) en niveles superiores.

## Qué se calcula en estas aplicaciones
- Fuerzas ficticias necesarias para describir la dinámica desde el marco acelerado.
- Comparación de resultados entre marcos inercial y no inercial.
- Diagnóstico de errores por mezcla de ecuaciones de diferentes marcos.

## Decisiones de diseño y operación basadas en este subtema
- Elegir el marco de referencia adecuado según el problema.
- Justificar el uso de fuerzas ficticias solo cuando el análisis lo requiere.
- Verificar la coherencia interna de las ecuaciones antes de operar numéricamente.

## Lectura didáctica
En este subtema, las aplicaciones reales se sostienen sobre la correcta identificación del marco de referencia y la justificación del uso de fuerzas ficticias. El objetivo no es solo obtener un número: es decidir si el modelo elegido describe bien el sistema físico y permite tomar decisiones técnicas con criterio.
