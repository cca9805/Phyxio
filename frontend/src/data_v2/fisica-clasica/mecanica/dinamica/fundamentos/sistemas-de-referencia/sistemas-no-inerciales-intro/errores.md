# Errores comunes

1) Pensar que la fuerza ficticia es una fuerza real (no es interacción).
2) Dibujar la fuerza ficticia en un DCL de un marco inercial (solo se usa dentro del marco no inercial).
3) Confundir “peso” con “normal”: el peso no cambia, la normal sí.
4) Olvidar el signo: Ff = -m aref (va opuesta a la aceleración del marco).
5) Mezclar marcos: usar ecuaciones de un marco y datos medidos en otro.
