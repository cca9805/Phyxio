# Modelos y validez

## Modelo base del subtema
Este subtema modela la dinámica de sistemas observados desde marcos no inerciales, donde aparecen fuerzas ficticias para mantener la coherencia matemática. Se parte de la comparación explícita entre la descripción inercial y la no inercial, usando variables definidas en cada sistema de referencia.

## Hipótesis operativas
- El marco no inercial es acelerado respecto a un marco inercial bien definido.
- Las fuerzas ficticias se introducen solo para describir la dinámica desde el marco acelerado.
- Se declaran explícitamente ejes, sentido positivo y origen antes de operar.

## Qué explica bien este modelo
- La aparición de fuerzas ficticias como consecuencia de la aceleración del marco.
- La equivalencia formal de resultados si se mantiene la coherencia interna en cada marco.
- El diagnóstico de errores al mezclar ecuaciones de diferentes marcos.

## Límites del modelo
- Si se mezclan ecuaciones de marcos distintos, los resultados pierden validez física.
- Si no se declaran correctamente los ejes y sentidos, los signos de las fuerzas pueden ser incorrectos.
- El modelo no es aplicable si el marco no inercial tiene aceleraciones variables no especificadas.

## Señales de que debes cambiar de modelo
- Resultados inconsistentes al comparar con la descripción inercial.
- Dificultad para justificar la aparición de fuerzas ficticias en el análisis.
- Incoherencias físicas persistentes aun con unidades y signos correctos.
