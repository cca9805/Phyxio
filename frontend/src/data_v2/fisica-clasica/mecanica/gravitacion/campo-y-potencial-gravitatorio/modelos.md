# Modelos y alcance

## Modelo base
- Masa fuente puntual o distribución esférica equivalente.
- Régimen clásico no relativista.
- Interacción gravitatoria tratada como conservativa.

## Hipótesis operativas
- Espacio continuo y diferenciable para usar \(\nabla V\).
- Referencia de potencial explícita (normalmente infinito).
- Ausencia de efectos relativistas significativos.

## Herramientas del subtema
- Campo radial \(\vec g=-GM\hat r/r^2\).
- Potencial \(V=-GM/r\).
- Relaciones \(\vec F=m\vec g\), \(U=mV\), \(\vec g=-\nabla V\).
- Superposición de fuentes para campo/potencial total.

## Qué explica bien
- Variación de \(g\) con distancia.
- Energía gravitatoria en sistemas ligados.
- Trabajo gravitatorio entre posiciones.

## Límites del modelo
- No incorpora curvatura del espacio-tiempo (relatividad general).
- Para geometrías complejas puede requerir integración numérica.
- En escalas cosmológicas no basta el marco newtoniano simple.
