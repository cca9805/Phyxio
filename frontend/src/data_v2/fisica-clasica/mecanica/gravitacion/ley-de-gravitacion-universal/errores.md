# Errores frecuentes

## 1- Usar mal la distancia \(r\)
Error típico: medir desde superficies en lugar de centros de masa.
Corrección: en gravitación newtoniana, \(r\) siempre se toma centro a centro.

## 2- Tratar la fuerza como escalar cuando el problema es vectorial
Error típico: calcular solo módulo y olvidar dirección/sentido.
Corrección: dibuja la recta que une masas y asigna signos en un eje radial.

## 3- Confundir \(g\) local con “constante universal”
Error típico: usar \(9.8\,\mathrm{m/s^2}\) en cualquier planeta/altura.
Corrección: si cambian \(M\) o \(r\), recalcula \(g=GM/r^2\).

## 4- Mezcla de unidades
Error típico: km con m, toneladas con kg.
Corrección: convierte todo a SI antes de sustituir.

## 5- Redondeo prematuro
Error típico: truncar demasiado en pasos intermedios.
Corrección: conserva al menos 3-4 cifras significativas hasta el final.

## 6- Ignorar dominio de validez
Error típico: exigir precisión extrema sin considerar límites del modelo clásico.
Corrección: identifica cuándo se requieren correcciones relativistas o métodos numéricos.
