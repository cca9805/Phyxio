
# Ejemplos guiados

## Ejemplo 1- Polea simple fija
**Enunciado:** Se desea elevar una caja de $m=10\ kg$ usando una polea fija. ¿Qué fuerza se debe aplicar?
**Pasos de resolución:**
1. Aplica $P=R=mg$.
2. Sustituye: $P=10\cdot 9.8=98$ N.
3. Interpreta: no hay ventaja mecánica, solo cambio de dirección.

## Ejemplo 2- Polea móvil
**Enunciado:** Se sostiene una carga de $m=20\ kg$ con una polea móvil. ¿Qué fuerza se aplica?
**Pasos de resolución:**
1. Aplica $P=\frac{R}{2}$.
2. Sustituye: $P=\frac{20\cdot 9.8}{2}=98$ N.
3. Interpreta: ventaja mecánica $VM=2$.

## Ejemplo 3- Polipasto de $n$ tramos
**Enunciado:** Se usa un polipasto de $n=4$ tramos para levantar $m=100\ kg$. ¿Qué fuerza se requiere?
**Pasos de resolución:**
1. Aplica $P=\frac{R}{n}$.
2. Sustituye: $P=\frac{100\cdot 9.8}{4}=245$ N.
3. Interpreta: ventaja mecánica $VM=4$.

## Ejemplo 4- Sistema real con fricción
**Enunciado:** Si la polea tiene fricción y requiere $10\ N$ extra para girar, ¿cuál es la fuerza total para elevar $m=10\ kg$?
**Pasos de resolución:**
1. Aplica $P=mg+10$.
2. Sustituye: $P=98+10=108$ N.
3. Interpreta: la fricción aumenta el esfuerzo necesario.

## Ejemplo 5- Polea compuesta
**Enunciado:** Se combinan dos poleas móviles y una fija para levantar $m=40\ kg$. ¿Qué fuerza se requiere si el sistema tiene $n=4$ tramos?
**Pasos de resolución:**
1. Aplica $P=\frac{R}{n}$.
2. Sustituye: $P=\frac{40\cdot 9.8}{4}=98$ N.
3. Interpreta: el análisis requiere identificar correctamente el número de tramos y el montaje (ver gráfico Coord).

## Criterio didáctico de cierre
En todos los ejemplos, la clave es justificar hipótesis, mantener consistencia de unidades y verificar que el resultado describe de forma coherente el comportamiento esperado del sistema de poleas.
# Poleas - Ejemplos guiados

## Ejemplo 1- Sistema ideal
Datos: \(R=900\ \text{N}\), \(n=3\).

$$
P=\frac{R}{n}=\frac{900}{3}=300\ \text{N}
$$

## Ejemplo 2- Sistema real
Datos: \(R=900\ \text{N}\), \(n=3\), \(\eta=0.80\).

$$
P=\frac{R}{n\eta}=\frac{900}{3\cdot 0.80}=375\ \text{N}
$$

Comparacion:
- Ideal: 300 N.
- Real: 375 N.

## Ejemplo 3- Relacion de recorridos
Si \(n=4\) y quieres elevar \(s_R=0.5\ \text{m}\):

$$
s_P=n\,s_R=4\cdot 0.5=2.0\ \text{m}
$$

Lectura: se ahorra fuerza, pero se tira mas cuerda.
