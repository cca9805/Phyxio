# Rendimiento Y Ventaja Mecanica - Aplicaciones

## Idea clave
Este tema permite comparar maquinas no solo por fuerza requerida, sino por eficiencia global.

## Aplicaciones
- Seleccion de mecanismos de izado.
- Evaluacion de herramientas manuales y ergonomia.
- Diagnostico de desgaste por caida de rendimiento.
- Optimizacion energetica de lineas mecanicas.

## Indicadores practicos
- \(VM_{real}\) para esfuerzo operativo.
- \(\eta\) para calidad energetica del mecanismo.
- Evolucion temporal de \(\eta\) para mantenimiento.
