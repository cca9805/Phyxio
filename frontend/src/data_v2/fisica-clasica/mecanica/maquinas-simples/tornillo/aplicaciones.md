# Tornillo - Aplicaciones

## Idea clave
El tornillo permite generar grandes fuerzas axiales con control fino y poco esfuerzo tangencial.

## Aplicaciones
- Tornillo de banco y prensas.
- Gatos mecanicos de vehiculos.
- Husillos de posicionamiento en maquinas herramienta.
- Mecanismos de ajuste fino en instrumentacion.

## Criterios tecnicos
- Seleccionar paso segun compromiso entre fuerza y velocidad de avance.
- Gestionar friccion con lubricacion adecuada.
- Verificar torque maximo de accionamiento.
