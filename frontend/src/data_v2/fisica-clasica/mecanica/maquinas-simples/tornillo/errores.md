

# Errores frecuentes

## 1- Error de identificación de parámetros
Usar el diámetro en vez del radio en las fórmulas, o interpretar mal el paso (avance por vuelta, no longitud total de la rosca).

## 2- Error de interpretación física
Omitir el factor $2\pi$ al pasar de recorrido circular a fuerza, o mezclar torque y fuerza ($\tau=F\,r$).

## 3- Error de unidades y parámetros
No incluir el factor de eficiencia $\eta$ en sistemas reales, o mezclar unidades de longitud y fuerza.

## 4- Error de dominio de validez
Aplicar el modelo ideal en sistemas con fricción significativa o materiales deformables.

## 5- Error de cierre de resolución
No visualizar una vuelta completa para distinguir $2\pi r$ frente a $p$, o calcular la fuerza usando la longitud total de la rosca.

## Cómo evitar estos errores de forma sistemática
- Usa siempre el radio, no el diámetro, y define correctamente el paso.
- Incluye el factor $2\pi$ y distingue entre torque y fuerza.
- Considera la eficiencia real y usa unidades consistentes.
- Visualiza el recorrido de una vuelta completa y revisa el modelo según el material y la fricción.
