# Tornillo - Errores comunes

## Error 1- Usar diametro como radio
En formulas entra \(r\), no \(d\). Si usas diametro duplicas error.

## Error 2- Omitir factor \(2\pi\)
Es uno de los fallos mas frecuentes al pasar de recorrido circular a fuerza.

## Error 3- Paso mal interpretado
El paso es avance por vuelta, no longitud total de rosca.

## Error 4- Ignorar rendimiento
En roscas reales la friccion puede cambiar mucho la fuerza necesaria.

## Error 5- Mezclar torque y fuerza
No son la misma magnitud: \(\tau=F\,r\).

## Recomendacion
Siempre dibuja una vuelta completa para visualizar \(2\pi r\) frente a \(p\).
