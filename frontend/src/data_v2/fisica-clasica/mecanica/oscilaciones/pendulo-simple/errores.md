# Errores frecuentes

## 1- Error de identificación de variables
Aplicar fórmula exacta de pequeño ángulo a amplitudes grandes.

## 2- Error de interpretación física
Usar longitud de cuerda sin incluir punto de suspensión al centro de masa.

## 3- Error de dominio de validez
Confundir periodo con semiperiodo.

## 4- Error de procedimiento algebraico
Tomar g constante en cualquier contexto sin justificar.

## 5- Error de unidades y parámetros
Mezclar grados y radianes en ecuación diferencial linealizada.

## 6- Error de cierre de resolución
Despreciar amortiguamiento cuando domina el experimento.

## Cómo evitar estos errores de forma sistemática
- Define variables y unidades antes de sustituir.
- Comprueba hipótesis del modelo antes de calcular.
- Cierra siempre con verificación física del resultado.
