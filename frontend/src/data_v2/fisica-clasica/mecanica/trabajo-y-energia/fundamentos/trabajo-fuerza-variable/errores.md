# Errores frecuentes

1) **Integrar respecto al tiempo en vez de posicion**
En este Tema, el trabajo variable se plantea sobre $x$: $W=\int F(x)dx$.

2) **Ignorar el signo de areas**
Area bajo el eje $x$ implica trabajo negativo.

3) **Usar Hooke sin intervalo**
No siempre es $\tfrac12 kx^2$; para dos posiciones es
$\tfrac{k}{2}(x_2^2-x_1^2)$.

4) **Confundir fuerza media con promedio aritmetico no justificado**
Debe ser consistente con el area real bajo la curva.
