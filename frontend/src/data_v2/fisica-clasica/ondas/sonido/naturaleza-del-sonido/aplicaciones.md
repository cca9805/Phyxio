# Naturaleza Del Sonido - Aplicaciones

## Idea clave
Entender la naturaleza fisica del sonido permite tomar decisiones tecnicas correctas antes de entrar a calculos avanzados de intensidad, espectro o propagacion compleja.

## 1- Donde se aplica de forma directa
- Diseno y calibracion de sistemas de audio.
- Acustica de salas, aulas y recintos industriales.
- Ultrasonido medico e industrial.
- Metrologia acustica y sensores de presion sonora.

## 2- Que decision tecnica permite este subtema
Con este subtema puedes decidir:
- si un problema requiere modelo mecanico longitudinal,
- que ecuacion usar para estimar velocidad en el medio,
- cuando usar aproximacion por temperatura del aire,
- como interpretar diferencias entre propagacion y oscilacion local.

## 3- Relacion con `Calculadora`, `Leyenda` y `Graficas`
1. En `Calculadora`, usa primero la formula minima para la incognita.
2. En `Leyenda`, verifica significado fisico y unidad SI de cada variable.
3. En `Graficas`, comprueba coherencia entre tendencia visual y resultado numerico.

## 4- Formulas base usadas en aplicaciones
- $$v=\lambda f$$
- $$T=\frac{1}{f}$$
- $$v=\sqrt{\frac{B}{\rho}}$$
- $$v\approx331+0.6T_c$$

## 5- Caso practico tipico
Objetivo: comparar propagacion en aire y agua para una misma frecuencia.

Resultado conceptual:
- \(f\) la fija la fuente,
- \(v\) cambia con el medio,
- por tanto \(\lambda\) cambia con el medio.

## 6- Cierre
Una aplicacion esta bien cerrada cuando formula, unidades, grafica e interpretacion del medio apuntan al mismo resultado fisico.
