# Modelo y validez

Las magnitudes físicas constituyen el **modelo básico de cuantificación** de la
realidad física.

Mediante la asignación de valores numéricos y unidades, los fenómenos naturales
pueden describirse, compararse y analizarse de forma objetiva.

Este modelo permite:
- construir leyes físicas
- realizar predicciones
- validar teorías mediante experimentación
