## ¿Qué es una magnitud física?

Una **magnitud física** es toda propiedad de un cuerpo o fenómeno que puede
medirse y expresarse cuantitativamente mediante un **número** y una **unidad**.

Ejemplos de magnitudes físicas son:
- la longitud
- la masa
- el tiempo
- la temperatura
- la velocidad
- la fuerza

---

## Magnitudes fundamentales y derivadas

### Magnitudes fundamentales
Son aquellas que se definen de manera independiente y constituyen la base de
cualquier sistema de unidades.

Ejemplos:
- longitud
- masa
- tiempo
- corriente eléctrica
- temperatura
- cantidad de sustancia
- intensidad luminosa

### Magnitudes derivadas
Se obtienen combinando magnitudes fundamentales mediante relaciones matemáticas.

Ejemplos:
- velocidad (longitud / tiempo)
- aceleración (longitud / tiempo²)
- fuerza (masa · aceleración)
- energía (fuerza · distancia)

---

## Unidades de medida

Una **unidad** es una cantidad de referencia que se utiliza para expresar el
valor de una magnitud física.

Toda medición se expresa como:

valor = número × unidad

Por ejemplo:
- 5 m
- 12 s
- 3,2 kg

---

## Sistema Internacional de Unidades (SI)

El **Sistema Internacional (SI)** es el sistema de unidades adoptado de forma
oficial en ciencia y tecnología para unificar las mediciones físicas.

Su uso garantiza:
- coherencia en los cálculos
- comunicación científica universal
- reducción de errores

---

## Importancia del uso correcto de unidades

El uso incorrecto de unidades puede invalidar completamente un resultado,
aunque el procedimiento matemático sea correcto.

En física:
- **no existen números "puros"**
- todo resultado debe llevar unidad
