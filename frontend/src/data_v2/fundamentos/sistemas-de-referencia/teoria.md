# sistemas-de-referencia · Teoría

## ¿Qué es un sistema de referencia?

Un **sistema de referencia** es el conjunto de criterios que utiliza un
observador para describir la posición y el movimiento de un objeto.

Incluye:
- un origen
- unos ejes de referencia
- un sistema de medida del tiempo

---

## El papel del observador

En física, la descripción de un fenómeno depende del observador que lo analiza.

Un mismo objeto puede:
- estar en reposo respecto a un observador
- estar en movimiento respecto a otro

Esto implica que **no existe el movimiento absoluto**, sino siempre relativo a
un sistema de referencia.

---

## Reposo y movimiento

Un cuerpo está:
- **en reposo** si su posición no cambia respecto al sistema de referencia
- **en movimiento** si su posición varía con el tiempo respecto a ese sistema

Ambos conceptos solo tienen sentido cuando se especifica el sistema de referencia.

---

## Sistemas de referencia habituales

Algunos sistemas de referencia comunes son:
- la superficie terrestre
- un vehículo en movimiento
- un sistema de ejes cartesianos ideal

La elección adecuada del sistema de referencia simplifica el análisis físico.
