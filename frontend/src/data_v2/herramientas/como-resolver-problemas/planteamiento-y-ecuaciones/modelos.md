# Modelo y validez (MCU)

## Suposiciones del modelo
- Trayectoria circular de radio constante.
- Velocidad angular constante.
- No se estudian fuerzas (solo cinemática).

## Dominio de validez
Apropiado cuando el movimiento se mantiene a radio fijo y con ritmo uniforme:
- ruedas, poleas, discos girando,
- satélites en órbitas casi circulares (a primera aproximación),
- ventiladores, carruseles, centrifugadoras.

## Cuándo NO aplicar MCU
- Si cambia la rapidez (entonces es MCUA).
- Si cambia el radio (movimiento curvilíneo no circular o espiral).
- Si hay paradas/arranques (ω no constante).

## Casos límite útiles
- Si ω = 0 → reposo angular.
- Si R aumenta manteniendo v, entonces ω disminuye (pero ya no es MCU puro si R varía).
