# Ejemplos (Encuentros y persecuciones)

## MRU vs MRU (persecución)
A sale desde xA0=0 con vA=6 m/s.
B sale desde xB0=-30 con vB=10 m/s, al mismo tiempo.

Encuentro:
6t = -30 + 10t  => 4t = 30 => t=7.5 s
x*=45 m

## Con retraso
Mismos datos, pero B sale 2 s tarde:
6t = -30 + 10(t-2) => 6t = -30 + 10t -20 => 4t=50 => t=12.5 s
(comprueba t>=2 ✅)
