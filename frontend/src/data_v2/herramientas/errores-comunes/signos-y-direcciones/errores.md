# Errores comunes (Encuentros y persecuciones)

## 1) Olvidar el retraso Δt
Si B sale tarde, su tiempo efectivo es (t-Δt), no t.

## 2) Aceptar soluciones no físicas
- t < 0: imposible
- t < Δt: B aún no había salido

## 3) Signos de velocidades
Define un eje y sé consistente.

## 4) No comprobar si realmente se alcanzan
Si vA >= vB (en persecución MRU) y A va delante, B no alcanza.
