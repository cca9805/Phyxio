# Modelo y validez (Encuentros y persecuciones)

## Suposiciones
- Movimiento 1D.
- MRU/MRUA por tramos.
- Retrasos de salida representados con Δt.

## Validez
Perfecto para problemas de examen y situaciones simplificadas.

## No aplicar
- Si hay cambios de velocidad no modelados (frenadas, curvas, tráfico…).
- Si el movimiento es 2D y no lo reduces por ejes.
