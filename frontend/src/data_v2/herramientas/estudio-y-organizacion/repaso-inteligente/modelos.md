# Modelo y validez (MCUA)

## Suposiciones del modelo
- Trayectoria circular con radio constante.
- Aceleración angular constante (α constante).
- No se modelan fuerzas (cinemática pura).

## Dominio de validez
Buena aproximación en fases de arranque/frenado “uniforme”:
- discos que aceleran de forma casi constante,
- ventiladores, platos giratorios, motores en régimen simple,
- sistemas donde el par neto es aproximadamente constante (ya en dinámica).

## Cuándo NO aplicar MCUA
- Si α cambia mucho con el tiempo.
- Si el radio cambia (no circular).
- Si hay saltos (aceleraciones bruscas).

## Casos límite
- Si α = 0 → se reduce a MCU.
- Si ω0 = 0 → arranque desde reposo angular.
