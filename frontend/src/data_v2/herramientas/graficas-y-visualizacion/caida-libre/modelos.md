# Modelo físico

En el modelo de caída libre se realizan las siguientes idealizaciones:

- El cuerpo se considera puntual.
- Se desprecia el rozamiento con el aire.
- La aceleración de la gravedad es constante.
- El movimiento ocurre cerca de la superficie terrestre.

Este modelo es válido para velocidades moderadas y alturas pequeñas comparadas con el radio de la Tierra.

## Limitaciones del modelo
- No es válido para objetos muy ligeros.
- Falla a grandes alturas.
- No describe trayectorias no verticales.

A pesar de estas limitaciones, el modelo describe con gran precisión numerosos fenómenos cotidianos.
