# Modelo y validez (MRU)

## Suposiciones del modelo
- El movimiento ocurre en una dimensión y en línea recta.
- La velocidad es constante: $a=0$.
- No se consideran cambios de dirección ni variaciones de velocidad.
- El sistema de referencia es inercial (no acelerado).

## Dominio de validez
El MRU es una buena aproximación cuando:
- el intervalo de tiempo es corto y la velocidad apenas cambia,
- o el móvil va a velocidad casi constante (ej. cinta transportadora, tramo llano a velocidad mantenida).

## Cuándo NO aplicar MRU
- Si hay aceleración apreciable (frenadas, arranques, caída libre).
- Si cambia la dirección (trayectoria curva).
- Si hay fuerzas netas no compensadas que cambian la velocidad.

## Casos límite útiles
- $v=0$ → reposo.
- $x_0=0$ → origen en la posición inicial.
- $t<0$ solo tendría sentido si estás describiendo “antes” del instante elegido como $t=0$; en problemas escolares se usa normalmente $t\ge 0$.
