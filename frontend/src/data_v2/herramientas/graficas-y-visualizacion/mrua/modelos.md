# Modelo y validez (MRUA)

## Suposiciones del modelo
- Movimiento unidimensional en línea recta.
- Aceleración constante ($a$ constante).
- No se consideran cambios bruscos de aceleración (tirones), ni fuerzas variables.

## Dominio de validez
MRUA es buena aproximación cuando:
- el intervalo de tiempo es corto,
- o la fuerza neta es aproximadamente constante.

Ejemplos típicos:
- frenada uniforme,
- arranque uniforme,
- caída libre ideal cerca de la superficie terrestre (si tomas $a=g$ constante).

## Cuándo NO aplicar MRUA
- Si $a$ cambia con el tiempo (motor variable, rozamiento muy cambiante).
- Si la trayectoria no es recta.
- Si hay colisiones o cambios repentinos de velocidad.

## Casos límite útiles
- Si $a=0$ → se reduce a MRU.
- Si $v_0=0$ → arranque desde reposo.
