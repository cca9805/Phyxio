# Modelo y validez (Tiro horizontal)

## Suposiciones del modelo
- El movimiento se analiza en **dos dimensiones** (horizontal y vertical).
- La **velocidad inicial solo tiene componente horizontal** ($v_{y0}=0$).
- La aceleración es constante y vertical: $a_x = 0$, $a_y = -g$.
- No se considera el rozamiento con el aire.
- El campo gravitatorio se supone uniforme.

---

## Dominio de validez
El modelo de tiro horizontal es una buena aproximación cuando:
- las velocidades no son muy grandes,
- la altura de lanzamiento es moderada,
- el rozamiento con el aire es despreciable frente al peso.

Ejemplos típicos:
- una pelota que sale del borde de una mesa,
- un objeto que cae desde una plataforma,
- lanzamientos a corta distancia en el laboratorio o el aula.

---

## Cuándo NO aplicar tiro horizontal
- Si el rozamiento con el aire es importante.
- Si el cuerpo tiene sustentación aerodinámica.
- Si la altura es tan grande que $g$ no puede considerarse constante.
- Si la velocidad inicial tiene componente vertical distinta de cero
  (en ese caso es **tiro parabólico**).

---

## Casos límite útiles
- Si $v_0 = 0$ → se reduce a **caída libre vertical**.
- Si $y_0 = 0$ → el tiempo de vuelo es nulo.
- Si se analiza solo el eje $x$ → se reduce a **MRU**.
