# Modelo y validez (Tiro parabólico)

## Suposiciones del modelo
Para describir el tiro parabólico se adoptan las siguientes hipótesis:

- El proyectil se trata como un **punto material**.
- No existe rozamiento con el aire.
- La aceleración de la gravedad es **constante** y vertical.
- El movimiento ocurre cerca de la superficie terrestre.
- El sistema de referencia es inercial.

---

## Dominio de validez
El modelo de tiro parabólico es una buena aproximación cuando:
- las velocidades no son muy grandes,
- las distancias recorridas no son extremas,
- el rozamiento del aire es despreciable frente al peso.

En estas condiciones, las ecuaciones predicen correctamente la trayectoria y los
tiempos del movimiento.

---

## Cuándo NO aplicar este modelo
El modelo deja de ser válido si:
- el rozamiento con el aire es importante,
- el proyectil tiene gran superficie o baja masa,
- se alcanzan grandes alturas (g deja de ser constante),
- intervienen fuerzas adicionales (empujes, motores, viento).

En estos casos es necesario usar modelos más complejos.

---

## Casos límite importantes
El tiro parabólico incluye como casos particulares:

- **θ = 0°**  
  → se reduce al **tiro horizontal**.

- **θ = 90°**  
  → se reduce al **lanzamiento vertical** (caída libre con v₀ ≠ 0).

Estos casos ayudan a comprobar la coherencia de las ecuaciones.

---

## Comprobaciones físicas
Al usar el modelo, conviene verificar:
- Si g → 0, el movimiento deja de ser parabólico.
- Si vy₀ = 0, no existe altura máxima.
- Si vx₀ = 0, no hay desplazamiento horizontal.

Estas comprobaciones sirven para detectar errores de planteamiento.
