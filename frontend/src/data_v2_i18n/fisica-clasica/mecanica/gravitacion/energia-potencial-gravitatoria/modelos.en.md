# Gravitational potential-energy models

## Ideal model

The ideal model assumes a point-like or spherically equivalent source mass [[M]], a test mass [[m]], and radial distance [[r]] measured from the source center. Under these assumptions, potential energy [[U]] is fully described by a conservative central interaction. The practical advantage is direct state comparison without solving full trajectory equations. This model therefore supports fast interpretation of energetic transitions and makes process-level reasoning transparent in exam and engineering-style contexts.

## Hypotheses

The operational hypotheses are explicit. First, classical gravity is assumed, with no dominant relativistic correction. Second, non-conservative losses are neglected when using compensation between [[DeltaU]] and [[DeltaK]]. Third, an explicit and stable reference level is required for [[U]], [[Ui]], [[Uf]], and [[Wg]]. Fourth, SI consistency is enforced for [[G]], [[M]], [[m]], [[ri]], and [[rf]]. Fifth, the process is represented through energetic states rather than detailed force-time microstructure.

## Quantitative validity domain

The general model is valid when [[ri]] > 0 and [[rf]] > 0 and source geometry supports central approximation. A practical quantitative threshold is to use local near-surface approximation only if h/r <= 0.01. If h/r > 0.01, the full radial model should be restored. Another practical criterion is estimated relative energetic error <= 5 percent for accepting linearization. If predicted error exceeds 5 percent, the model should be escalated to the general expression.

## Model failure signals

Failure appears when signs contradict process physics, for example outward transfer with negative [[DeltaU]] without justified reference change. Failure also appears when local height and absolute radius are mixed without conversion. A third failure mode is treating [[g]] as constant in wide radial ranges. In these cases, coherence between [[Wg]] and [[DeltaU]] usually breaks first. That break is a diagnostic signal that assumptions are no longer valid.

## Extended or alternative model

When geometry becomes complex or precision demands increase, an extended model is required. For small terrestrial displacement, the local model with [[h]] and [[g]] is efficient. For orbital-range transitions, the robust choice is the full radial model with explicit state classification through [[E]]. For multi-source environments, numerical or effective-potential approaches become preferable. Model switching is not stylistic; it is driven by scale ratio, required uncertainty, and physical architecture.

## Operational comparison

Ideal radial model is fast and highly interpretable for most educational tasks. Local model is even faster for small displacements but fragile outside validity limits. Extended models are computationally heavier but necessary in high-precision or multi-body settings. Operational rule is straightforward: if h/r <= 0.01 and quick estimate is acceptable, local model is efficient. If h/r > 0.01 or rigorous traceability is needed, use the full radial model. If multiple dominant sources exist, switch to extended modeling.
