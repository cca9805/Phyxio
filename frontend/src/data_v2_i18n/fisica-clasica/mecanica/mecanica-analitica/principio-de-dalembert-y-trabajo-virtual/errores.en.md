## Conceptual errors

### Error 1: Believing that an eliminated force does not exist

**Why it seems correct**  
If the normal force or tension does not appear in the reduced equation, it seems to have been neglected.

**Why it is incorrect**  
The force may exist while performing no virtual work on compatible displacements.

**Detection signal**  
The student says that the guide exerts no normal force because the angular equation does not contain it.

**Conceptual correction**  
Distinguish physical existence from relevance in the virtual balance.

## Model errors

### Error 2: Using incompatible virtual displacements

**Why it seems correct**  
A small displacement is drawn, and any small displacement seems acceptable.

**Why it is incorrect**  
[[dr]] must respect the instantaneous constraints. Otherwise [[Wv]] does not correspond to the system.

**Detection signal**  
The variation crosses a guide, stretches an inextensible string, or changes a fixed distance.

**Conceptual correction**  
Build admissible variations from geometry first.

## Mathematical errors

### Error 3: Always measuring [[Qj]] in newtons

**Why it seems correct**  
It is called generalized force and is mentally associated with an ordinary force.

**Why it is incorrect**  
The unit of [[Qj]] depends on the unit of [[qi]]. In an angular coordinate, it is usually torque.

**Detection signal**  
Newtons and newton meters are mixed without checking dimensions.

**Conceptual correction**  
Read [[Qj]] as a virtual-work coefficient.

## Interpretation errors

### Error 4: Confusing virtual work with real work

**Why it seems correct**  
Both use force times displacement.

**Why it is incorrect**  
Real work occurs along a time trajectory. Virtual work is an instantaneous compatible test.

**Detection signal**  
[[Wv]] is integrated as if it described energy transferred during real motion.

**Conceptual correction**  
Separate virtual variation, real displacement, and time evolution.

## Quick self-control rule

Before using D'Alembert, check: 1) [[dr]] is compatible; 2) eliminated reactions are ideal; 3) [[F]], [[m]], and [[a]] belong to the same element; 4) [[Qj]] has the correct dimension for [[qi]]; 5) [[Ri]] is set equal to [[Qj]] only when the generalized force is well defined.
