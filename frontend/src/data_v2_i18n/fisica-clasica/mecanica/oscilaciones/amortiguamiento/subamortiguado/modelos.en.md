## Ideal model

The ideal underdamped model is a free linear second-order oscillator with equivalent viscous damping. The output [[x]] oscillates around equilibrium while its amplitude is bounded by [[x_env]]. The condition [[gamma]] < [[omega0]] guarantees that a real damped angular frequency [[omega_d]] exists.

## Hypotheses

Effective mass, stiffness, and damping remain constant. The amplitude [[A]] is small enough not to activate nonlinearities, stops, or stiffness changes. Dissipation is proportional to velocity, and no dominant periodic external force is present. Time [[t]] is measured from the beginning of the transient, and phase [[phi]] summarizes the initial state.

## Quantitative validity domain

The minimum criterion is 0 < [[gamma]]/[[omega0]] < 1. If the ratio is below 0.1, the system resembles an almost undamped oscillator for several cycles. Between 0.1 and 0.7, a clear oscillation with a decaying envelope is usually visible. Near 1, [[omega_d]] becomes small and the system approaches the critical boundary.

The dimensional check requires [[gamma]], [[omega0]], and [[omega_d]] to be compatible inverse-time quantities, [[gamma]][[t]] to be dimensionless, and [[A]], [[x_env]], and [[x]] to share the same unit.

## Model failure signals

The model fails if maxima do not follow an approximately exponential envelope, if the period changes irregularly, if amplitude grows without external input, or if the curve shows impacts against stops. It also fails if an external force maintains the oscillation, because the response is no longer free.

## Extended or alternative model

When dissipation depends on speed or amplitude, a nonlinear model or experimental envelope fitting is more appropriate. If periodic forcing is present, the forced oscillator model should be used. If [[gamma]] is too close to [[omega0]], the critical model may be a better practical description within uncertainty.

## Operational comparison

The underdamped model is appropriate when decreasing cycles are observed and the goal is to separate rhythm from amplitude loss. The alternative model is justified when the curve no longer resembles an oscillation enclosed by an exponential. The decision is made from signs, units, graph shape, and experimental tolerance.

It is advisable to change model when peak reduction cannot be fitted with an approximately constant rate, when the period changes from cycle to cycle without a controlled cause, or when the system receives external energy. In those cases, insisting on the ideal model hides relevant physical information.

In other words, when to switch to an alternative model is exactly when the measured curve no longer separates rhythm and decay cleanly. This practical criterion avoids forcing an underdamped reading onto data governed by different physics.
