## Conceptual errors

### Error
Treating [[W]] as if it were just force magnitude.

Why it seems correct
Everyday language often mixes effort and work, so students map large force directly to large transfer.

Detection signal
The solution discusses [[F]] only and omits [[d]] or [[theta]] in interpretation.

Mini-contrast example
Two interactions with the same [[F]] can yield very different work if orientation or displacement differs.

Conceptual correction
Work is energy transfer by force through displacement, not force by itself.

### Error
Assuming zero work means no physical effect.

Why it seems correct
A zero value is read as full physical inactivity.

Detection signal
The answer ignores directional effects or constraints even when transfer is zero.

Mini-contrast example
A perpendicular force may do near-zero work while still steering the motion.

Conceptual correction
Separate energetic transfer from directional dynamics.

## Model errors

### Error
Using constant-force work when force clearly varies across the segment.

Why it seems correct
The formula is simple and familiar, so it is applied beyond its domain.

Detection signal
Segment subdivision yields inconsistent totals or observations disagree systematically.

Mini-contrast example
Spring-like interactions require variable-force treatment, not one fixed force value.

Conceptual correction
Switch models when variation is significant.

### Error
Changing system boundary mid-solution.

Why it seems correct
Multi-body problems tempt silent shifts in what is considered the system.

Detection signal
Contribution-level [[W]] is interpreted as whole-system [[Wnet]] without justification.

Mini-contrast example
Work on a crate is not automatically work on crate plus operator.

Conceptual correction
Fix system definition first and keep it unchanged.

## Mathematical errors

### Error
Dismissing the sign of work contributions.

Why it seems correct
Students prioritize magnitude and treat sign as optional convention.

Detection signal
Unsigned summation replaces signed balance.

Mini-contrast example
Braking contribution reported as positive erases extraction reading.

Conceptual correction
Keep signed contributions and justify sign through [[theta]].

### Error
Mixing incompatible units for [[F]] or [[d]].

Why it seems correct
Arithmetic still returns a number.

Detection signal
Output scale is implausible for the scenario.

Mini-contrast example
Unconverted centimeter displacement distorts [[W]] by large factors.

Conceptual correction
Normalize SI units before substitution and run plausibility checks.

## Interpretation errors

### Error
Ending with a number and no causal reading.

Why it seems correct
Exam habits reward final value over physical explanation.

Detection signal
No statement connects sign, geometry, and expected motion effect.

Mini-contrast example
Equal absolute values can represent opposite physical stories depending on sign and context.

Conceptual correction
Finish with one sentence linking result, orientation, and energetic consequence.

## Quick self-control rule

1. Fix system and segment.
2. Check [[F]], [[d]], and [[theta]] in one consistent frame.
3. Predict sign before computation.
4. Sum signed contributions for [[Wnet]] when needed.
5. Validate order of magnitude and physical coherence.
