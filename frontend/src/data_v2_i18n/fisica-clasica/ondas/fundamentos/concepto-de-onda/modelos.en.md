## Ideal model

The main model is the wave as a propagating disturbance. It represents a local state [[y]] that changes with [[x]] and [[t]], with amplitude [[A]] and, when periodicity exists, with [[lambda]], [[f]], [[T]], and [[v]].

The central simplification is separating pattern advance from local motion of the medium. The model preserves the identity of the disturbance and leaves out friction, strong dispersion, medium breakdown, and appreciable net transport of matter.

## Hypotheses

- The disturbance is identifiable and is transmitted from one point to another.
- The medium or field allows a defined propagation speed.
- Oscillations are small enough for stable quantities to be used.
- If periodicity is used, the pattern repeats regularly.

If the first assumption fails, there is no clearly recognizable wave. If the second fails, speed is no longer a simple reading. If the third fails, the profile may deform and the ideal model loses predictive power.

## Quantitative validity domain

The approximation is reasonable for amplitudes below 10 % of the characteristic length of the setup and for speed variations below 5 % inside the observed region. On a classroom string, a typical order of magnitude is centimeters for [[A]] and decimeters or meters for [[lambda]].

It is also reasonable when several crests allow periodicity to be measured stably. If there is only an isolated pulse, propagation may still be discussed, but [[lambda]], [[f]], or [[T]] should not be forced as if the pattern were repetitive.

## Model failure signals

A clear failure signal is that the shape changes width as it advances. Another one is that amplitude decreases rapidly although attenuation was not part of the problem. It also fails if a mark of the medium shows dominant net drift instead of local oscillation.

In graphs, failure appears when no stable spatial repetition can be identified or when two successive measurements give incompatible values of [[v]]. In those cases, the real wave needs a richer model.

## Extended or alternative model

An extended model includes attenuation, dispersion, reflections, superposition, or transported energy. The concrete difference is that the shape is no longer described only by an ideal repeating disturbance: it may change with distance, combine with other waves, or depend on medium properties.

When to switch to the extended model: if the profile deforms, if amplitude falls systematically, if echoes appear, or if the goal is to calculate energy. For a signal measured at one detector only, a local model centered on [[y]], [[A]], [[f]], and [[T]] may be used.

## Operational comparison

| Reading | Propagating model | Local signal model |
|---|---|---|
| Main question | How the pattern advances | How one point changes |
| Key quantities | [[lambda]], [[f]], [[v]] | [[y]], [[A]], [[T]] |
| Risk | Confusing pattern and matter | Forgetting spatial structure |

The propagating model is better when the statement mentions crests, distance between repetitions, or advance speed. The local model is better when the problem focuses on a sensor, a string mark, or an oscillation at one point. The comparison prevents using every quantity at once without knowing which physical reading is needed.
