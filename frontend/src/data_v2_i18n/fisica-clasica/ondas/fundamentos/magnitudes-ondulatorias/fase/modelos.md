## Modelo ideal

El modelo ideal de fase describe una onda armónica con [[k]], [[omega]] y [[phi0]] constantes. En ese contexto, [[phi]] etiqueta el estado interno del ciclo y [[Delta_phi]] compara dos lecturas sin depender de la amplitud.

La onda se considera suficientemente estable para que una cresta, un valle o un cruce por equilibrio puedan seguirse como estados equivalentes.

## Hipótesis

La primera hipótesis es periodicidad clara: el patrón debe repetirse de forma reconocible. La segunda es homogeneidad aproximada del medio, de modo que [[k]] no cambie de una zona a otra durante la lectura. La tercera es estabilidad temporal, para que [[omega]] sea una medida útil del ritmo de fase.

También se fija una convención de signo. Sin esa convención, decir que una señal adelanta o retrasa a otra queda incompleto.

## Dominio de validez cuantitativo

El modelo funciona bien cuando varias oscilaciones consecutivas mantienen separación y periodo con variaciones pequeñas, por ejemplo menores del 5 % en una observación de laboratorio. En sonido estacionario, una ventana de varias decenas de ciclos suele bastar para estimar fase con sentido físico.

Si el pulso dura menos de unas pocas oscilaciones, hablar de una única fase global se vuelve débil. En ese caso conviene pasar a una descripción por componentes o por fase local.

## Señales de fallo del modelo

El modelo falla si el patrón se deforma, si aparecen varias frecuencias comparables o si la amplitud se atenúa tanto que la fase deja de poder leerse. También falla si se comparan señales de fuentes distintas sin sincronización común.

Una señal clara de fallo es que dos métodos razonables de estimar [[Delta_phi]] den lecturas incompatibles. Eso indica que no hay una única fase estable para el fenómeno observado.

## Modelo extendido o alternativo

Cuando conviene pasar al modelo extendido: si hay dispersión, ruido, pulsos cortos o mezcla de frecuencias, se usa una descomposición espectral o una fase instantánea definida por una componente dominante. En interferencia real, además se añaden amplitudes relativas y coherencia.

El modelo extendido no elimina la fase; la hace dependiente de la frecuencia, del tiempo o de la posición.

## Comparación operativa

El modelo ideal es adecuado para ondas sinusoidales limpias y para ejercicios donde [[k]] y [[omega]] están bien definidos. El modelo extendido es necesario cuando el patrón cambia durante la observación o cuando la pregunta depende de la envolvente, del ruido o de varias componentes simultáneas.

Operativamente, empieza por el modelo ideal si la gráfica muestra repetición regular y los datos dan una sola frecuencia. Cambia al modelo extendido si una misma medida no puede describirse con un [[Delta_phi]] estable.
