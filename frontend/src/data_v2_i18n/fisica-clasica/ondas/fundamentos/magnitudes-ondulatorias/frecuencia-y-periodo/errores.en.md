## Conceptual errors

### Error 1: confusing frequency with propagation speed

**Why it seems correct:** both seem to speak about speed.

**Why it is incorrect:** [[f]] counts cycles per second at one point; propagation speed describes spatial advance.

**Detection signal:** someone says that a higher-frequency wave necessarily travels faster.

**Correction:** separate temporal rhythm from motion through the medium.

**Contrast mini-example:** two sounds with different frequency travel at almost the same speed in air.

### Error 2: believing that period and frequency grow together

**Why it seems correct:** both describe repetition.

**Why it is incorrect:** if one cycle takes longer, fewer cycles fit into one second.

**Detection signal:** a long period is interpreted as high frequency.

**Correction:** remember that [[T]] and [[f]] are inverse readings.

**Contrast mini-example:** a slow pendulum has large period and small frequency.

## Model errors

### Error 3: using one frequency for an irregular signal

**Why it seems correct:** something resembling cycles can always be counted.

**Why it is incorrect:** if rhythm changes strongly, one [[f]] hides the variation.

**Detection signal:** different windows give incompatible results.

**Correction:** use average frequency by intervals or spectral analysis.

**Contrast mini-example:** a sustained note admits clear frequency; a rising siren does not.

## Mathematical errors

### Error 4: mixing Hz and rad/s

**Why it seems correct:** both units have inverse-time dimension.

**Why it is incorrect:** Hz counts cycles per second and rad/s measures angular phase per second.

**Detection signal:** [[omega]] is replaced with a value in Hz without conversion.

**Correction:** distinguish [[f]] from [[omega]] before calculating.

**Contrast mini-example:** 1 Hz corresponds to one full turn of phase per second.

## Interpretation errors

### Error 5: counting cycles incorrectly

**Why it seems correct:** one isolated crest looks like a complete cycle.

**Why it is incorrect:** a complete cycle requires returning to the same oscillation state.

**Detection signal:** [[N]] does not match the visible complete repetitions.

**Correction:** fix a start and end criterion for the cycle.

**Contrast mini-example:** crest to crest is one cycle; crest to trough is half a cycle.

## Quick self-control rule

Before solving, ask whether you are measuring cycles per second, duration of one cycle, or angular phase advance. Then check that [[N]] and [[Delta_t]] describe the same window and that the rhythm is stable.
