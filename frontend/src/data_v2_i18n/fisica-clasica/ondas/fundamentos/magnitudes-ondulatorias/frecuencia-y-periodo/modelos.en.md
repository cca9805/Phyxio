## Ideal model

The ideal model describes a periodic wave with constant temporal rhythm. In that case, [[f]] and [[T]] are stable inverses and [[omega]] expresses the same rhythm as angular phase advance.

The wave can be observed at one fixed point of the medium: it is not yet necessary to know how it travels through space.

## Hypotheses

The first hypothesis is that cycles repeat with nearly constant duration. The second is that what counts as a complete cycle is clearly identified. The third is that the window [[Delta_t]] contains enough repetitions for the count [[N]] to be meaningful.

The measuring instrument is also assumed not to alter the observed rhythm.

## Quantitative validity domain

The model works well if period variation between cycles is below 5 % during measurement. For manual counts, at least 5 complete cycles should be observed; with fewer cycles, an error of one crest changes the estimated frequency strongly.

If the signal has 100 Hz, the period is of order 0.01 s. If it has 1 Hz, the period is of order 1 s. Those scales help choose a reasonable observation window.

## Model failure signals

The model fails when rhythm changes during observation, when isolated pulses appear, or when several frequencies mix with comparable amplitudes. It also fails if the count includes half cycles without saying so.

A clear failure signal is that two different windows give very different frequencies without a recognized physical cause.

## Extended or alternative model

When to switch to the extended model: if the signal changes with time, use instantaneous frequency, windowed average frequency, or spectral analysis. In musical signals, damped vibrations, and pulses, one [[f]] may not describe the whole phenomenon.

The extended model keeps the idea of rhythm, but allows that rhythm to depend on time or on the analyzed component.

## Operational comparison

The ideal model is enough for stable sinusoidal waves, period exercises, and regular counts. The extended model is needed if the pattern deforms, if rhythm is not constant, or if the question depends on several components.

Operationally, start by measuring [[N]] in [[Delta_t]]. If the result stays stable when the window changes, use the ideal model; if it changes strongly, describe the signal by intervals.

The practical comparison is simple: a stable signal admits one pair [[f]] and [[T]]; a changing signal must state when it was measured and over which interval.

This distinction matters in reports. Giving a frequency without the observation window can be misleading when the signal evolves during the measurement.
