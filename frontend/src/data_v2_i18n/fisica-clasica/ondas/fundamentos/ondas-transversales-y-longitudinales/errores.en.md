## Conceptual errors

### Error 1: thinking transverse means vertical

**Why it seems correct:** many drawings show the wave advancing horizontally and the disturbance moving up and down. Visual memory turns a drawing convention into a false definition.

**Why it is incorrect:** transverse means perpendicular to the propagation direction. If advance is drawn tilted, the transverse component changes orientation too.

**Detection signal:** the student classifies without mentioning the propagation direction.

**Correction:** first fix the advance direction; then decide whether [[u_perp]] or [[u_parallel]] dominates.

**Contrast mini-example:** a vertically drawn string can still have a transverse wave if the disturbance is perpendicular to its advance.

## Model errors

### Error 2: forcing pure types in mixed waves

**Why it seems correct:** binary classification feels simpler than speaking of dominance. This is excessive simplification.

**Why it is incorrect:** some waves have relevant transverse and longitudinal components. If [[C_T]] and [[C_L]] are comparable, one label hides information.

**Detection signal:** the observation shows lateral displacements and compressions, but the answer chooses only one word.

**Correction:** use component dominance and state whether the mode is mixed.

**Contrast mini-example:** in surface water waves, particles do not move only up and down or only forward and backward.

## Mathematical errors

### Error 3: normalizing with different amplitudes

**Why it seems correct:** since all magnitudes are lengths, any quotient seems usable. The error comes from ignoring that the scale must be common.

**Why it is incorrect:** [[C_T]] and [[C_L]] are comparable only if they use the same [[A]] and the same wave state.

**Detection signal:** the student uses an amplitude from one setup and a component from another.

**Correction:** measure [[A]], [[u_perp]], and [[u_parallel]] in the same record.

**Contrast mini-example:** comparing a sideways string with a different spring compression may give indices with correct units but no common meaning.

## Interpretation errors

### Error 4: thinking longitudinal carries no disturbance

**Why it seems correct:** in a longitudinal wave, particles move back again and it seems that nothing advances. Local motion is confused with propagation.

**Why it is incorrect:** the compression does advance. The medium oscillates around equilibrium, but the compression pattern moves.

**Detection signal:** the student says sound is not a wave because no transverse crests are visible.

**Correction:** look for compressions, rarefactions, or pressure changes as the longitudinal disturbance.

**Contrast mini-example:** a microphone detects sound even though it sees no string moving up and down.

## Quick self-control rule

Before classifying, answer three questions. What is the propagation direction, which disturbance component dominates, and whether the scale [[A]] is common for comparison.

If the answer identifies neither [[u_perp]] nor [[u_parallel]], there is not yet a physical classification. If both components are relevant, use mixed mode or dominance instead of a pure label.

The final practical rule is to compare before naming. A quick word can hide the important data: which component of the medium moves and relative to which direction that motion was defined.

If the propagation direction is missing from the explanation, the classification is incomplete. If the component measurement is missing, the answer is only a visual guess, not physics.
