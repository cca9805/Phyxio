# Errors

## Conceptual errors

A common conceptual error is using a noninteger [[n]] as if it were a natural mode. This may look acceptable because algebra can return a decimal value, but a fixed-fixed string needs complete antinodes and nodes at both ends. An intermediate value may describe forced motion, not a clean standing mode.

Another error is believing that a standing string contains no propagation. The pattern does not travel, but it is sustained by two opposite traveling waves with speed [[v]]. Without that propagation there would be no energy exchange and no temporal selection of [[f_n]].

## Model errors

A frequent model error is using the total string length, including inactive parts beyond the supports, as [[L]]. The model needs the vibrating span between effective nodes. In an instrument, a fret or a bridge changes that span even if the material string is the same.

It is also unsafe to ignore whether [[T]] is nearly uniform. If the amplitude is large or the string slackens during motion, the frequency computed with the ideal model is no longer reliable. For stiff or very thick strings, high modes require an extended model.

## Mathematical errors

The most common mathematical failure is treating the dependence on [[T]] as direct proportionality. Tension enters through the speed relation, so doubling tension does not double [[v]] or [[f_n]]. Another failure is using total mass instead of [[mu]]. Linear density means mass distributed per metre.

Unit conversion is equally important. A [[mu]] given in grams per metre must be converted coherently before calculation. If the gram value is kept as if it were kilograms, [[v]] becomes too small and the whole set of [[f_n]] shifts.

## Interpretation errors

A numerical [[f_n]] is not enough. One must read which variable dominates the change: [[L]] through geometry, [[T]] through pull, [[mu]] through inertia, or [[n]] through modal selection. Without that reading, two similar frequencies may have opposite physical causes.

Another interpretation error is identifying [[lambda_n]] with [[L]] in every mode. That occurs only in one particular mode. In the fundamental, the string contains half a modal wavelength; in higher modes it contains several half-wavelengths. The drawing must support the interpretation.

## Quick self-control rule

Before accepting the result, check four points: [[L]] is the vibrating span, [[T]] and [[mu]] are positive and reasonable, [[n]] is an integer, and the drawing shows exactly the expected antinodes. If one check fails, [[f_n]] may be algebraically computed but physically misread.
