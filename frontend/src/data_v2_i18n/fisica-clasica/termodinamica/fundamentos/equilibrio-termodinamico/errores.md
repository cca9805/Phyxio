# Errores comunes en equilibrio termodinámico

## Errores conceptuales

### Error 1: "Equilibrio significa que nada se mueve"

**Descripción.** El alumno imagina un sistema congelado microscopicamente.

**Por que parece correcto.** En lenguaje cotidiano, equilibrio se asocia con reposo.

**Senal de deteccion.** Se niega el movimiento molecular o se dice que las particulas quedan quietas.

**Correccion conceptual.** Equilibrio significa ausencia de cambio macroscopico neto. Las particulas siguen moviendose, pero las variables de estado permanecen constantes.

**Mini-ejemplo de contraste.** Un gas en equilibrio tiene moleculas moviendose; aun asi [[T]] y [[P]] macroscopicas son estables.

### Error 2: "Temperatura uniforme basta para equilibrio completo"

**Descripción.** Se declara equilibrio solo porque [[DeltaT]] es cero.

**Por que parece correcto.** El equilibrio térmico es el caso más familiar.

**Senal de deteccion.** Se ignoran [[DeltaP]], composicion o reacciones posibles.

**Correccion conceptual.** El equilibrio completo exige revisar todos los procesos permitidos: calor, movimiento de fronteras, difusion y reaccion.

**Mini-ejemplo de contraste.** Dos gases pueden tener la misma [[T]] y distinta [[P]]; si un pistón se mueve, no hay equilibrio mecánico.

## Errores de modelo

### Error 3: "Estado estacionario es lo mismo que equilibrio"

**Descripción.** Se toma cualquier sistema constante en el tiempo como equilibrio.

**Por que parece correcto.** En ambos casos las variables pueden no cambiar temporalmente.

**Senal de deteccion.** Hay flujos permanentes a traves del sistema.

**Correccion conceptual.** Un estado estacionario puede sostener corrientes internas. El equilibrio termodinámico exige ausencia de flujos netos internos.

**Mini-ejemplo de contraste.** Una barra con calor entrando por un extremo y saliendo por otro es estacionaria, pero no esta en equilibrio.

## Errores matemáticos

### Error 4: "Usar igualdad exacta sin tolerancia experimental"

**Descripción.** Se exige que dos sensores den exactamente el mismo valor.

**Por que parece correcto.** Las formulas de equilibrio se escriben con igualdad.

**Senal de deteccion.** Se rechaza equilibrio por diferencias menores que la incertidumbre de medida.

**Correccion conceptual.** En laboratorio, equilibrio se evalua dentro de tolerancias. Lo relevante es si [[DeltaT]] o [[DeltaP]] son físicamente apreciables.

**Mini-ejemplo de contraste.** Dos lecturas de 300.00 K y 300.02 K pueden ser indistinguibles si el sensor tiene incertidumbre de 0.05 K.

## Errores de interpretacion

### Error 5: "Equilibrio siempre se alcanza rápido"

**Descripción.** Se supone que el sistema llega al equilibrio en cuanto se conecta o se deja aislado.

**Por que parece correcto.** Muchos ejemplos escolares idealizan la relajación.

**Senal de deteccion.** No se considera el tiempo de relajación [[tau]].

**Correccion conceptual.** El equilibrio requiere una escala temporal suficiente. Si [[tau]] es grande, el sistema puede parecer estable sin estar equilibrado.

**Mini-ejemplo de contraste.** Un bloque grueso puede tardar horas en igualar temperatura interna aunque su superficie parezca estable.

## Regla de autocontrol rápido

Antes de declarar equilibrio, pregunta: que intercambios permite la frontera, que gradientes quedan y en que tiempo se observan. Si queda [[DeltaT]], [[DeltaP]] o [[A]] relevante, no hay equilibrio completo.

Una forma práctica de evitar errores es separar tres frases. Primero: "el sistema no cambia apreciablemente durante mi observación". Segundo: "no detecto gradientes que puedan producir flujo". Tercero: "las restricciones permiten o impiden cada intercambio". Solo la combinacion de las tres justifica llamar equilibrio al estado. Si solo se cumple la primera, probablemente se trata de un transitorio lento o de un regimen estacionario.
