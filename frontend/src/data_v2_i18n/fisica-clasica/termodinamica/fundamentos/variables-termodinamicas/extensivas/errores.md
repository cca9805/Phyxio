# Errores comunes en variables extensivas

## Errores conceptuales

### Error 1: "Extensiva significa grande"

**Descripción.** El alumno cree que una magnitud es extensiva si su valor numerico es grande.

**Por que parece correcto.** La palabra extensiva sugiere extension o tamano.

**Senal de deteccion.** Se clasifica una presión alta como extensiva o una masa pequena como no extensiva.

**Correccion conceptual.** Extensiva significa proporcional al tamano del sistema, no grande en valor absoluto.

**Mini-ejemplo de contraste.** Un miligramo de muestra tiene masa [[m]] extensiva aunque sea pequena; una presión enorme sigue siendo intensiva.

### Error 2: "Toda magnitud calculada con masa es extensiva"

**Descripción.** Se clasifica [[rho]] como extensiva porque aparece [[m]] en su formula.

**Por que parece correcto.** La masa es el ejemplo más familiar de extensividad.

**Senal de deteccion.** Se dice que la densidad se duplica al duplicar la muestra.

**Correccion conceptual.** Un cociente de extensivas puede eliminar la escala. Si [[m]] y [[V]] se duplican juntos, [[rho]] queda constante.

**Mini-ejemplo de contraste.** Dos litros de agua tienen doble masa que uno, pero la misma densidad aproximada.

## Errores de modelo

### Error 3: "Sumar propiedades especificas como si fueran totales"

**Descripción.** Se suman valores de [[x_esp]] de dos muestras para obtener un total.

**Por que parece correcto.** La palabra específica puede parecer solo una forma alternativa de escribir la misma propiedad.

**Senal de deteccion.** Se calcula energía específica total sumando kJ/kg de dos subsistemas.

**Correccion conceptual.** Primero recupera la extensiva total multiplicando por masa; después suma totales.

**Mini-ejemplo de contraste.** Dos muestras con 10 kJ/kg no dan 20 kJ/kg al juntarse; siguen en 10 kJ/kg si son equivalentes.

## Errores matemáticos

### Error 4: "Aplicar aditividad con referencias incompatibles"

**Descripción.** Se suman energías internas o entropias calculadas con referencias distintas.

**Por que parece correcto.** La formula de aditividad parece autorizar cualquier suma de valores.

**Senal de deteccion.** Dos tablas o referencias de cero se mezclan sin conversion.

**Correccion conceptual.** Las extensivas se suman si representan la misma magnitud con la misma referencia.

**Mini-ejemplo de contraste.** No se suma una energía medida desde un cero arbitrario con otra definida desde un cero distinto sin ajustar referencias.

## Errores de interpretacion

### Error 5: "Comparar muestras por extensivas sin normalizar"

**Descripción.** Se concluye que una muestra esta más caliente o es más energética por tener mayor [[U]] total.

**Por que parece correcto.** Mayor energía total parece indicar mayor intensidad energética.

**Senal de deteccion.** Una muestra grande se declara más caliente que una pequena solo por tener mayor energía interna.

**Correccion conceptual.** Para comparar estados de muestras de distinto tamano hay que usar magnitudes normalizadas como [[x_esp]] o variables intensivas.

**Mini-ejemplo de contraste.** Una piscina tibia puede tener más [[U]] total que una taza de cafe caliente, pero no esta a mayor temperatura.

## Regla de autocontrol rápido

Antes de clasificar una magnitud, imagina dos copias identicas del sistema. Si el valor total se duplica, es extensiva. Si queda igual, probablemente es intensiva o normalizada. Si cambia de forma no lineal, revisa mezcla, superficie o interacciones.
