# Aplicaciones fisicas

## 1. Laboratorio escolar

Aparece en toda medida con regla, cronometro, balanza o sensor. Se calcula que cifras conservar y que incertidumbre declarar. El resultado significa precision honesta. El error peligroso es escribir muchos decimales y aparentar una exactitud que el instrumento no tiene. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 2. Cinematica experimental

Aparece al medir distancias y tiempos para obtener velocidades o aceleraciones. Se calcula como la incertidumbre de cada dato afecta el resultado. El resultado indica si una diferencia es real o compatible con error de medida. El error peligroso es comparar numeros sin incertidumbre. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 3. Dinamica

Aparece al calcular fuerzas desde masa y aceleracion medidas. Se estima la incertidumbre de la fuerza. El resultado permite decidir si dos fuerzas son distinguibles. El error peligroso es redondear demasiado pronto y perder informacion util. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 4. Electricidad

Aparece al medir tension, corriente y resistencia. Se calcula si una ley como V = I R se cumple dentro de tolerancias. El resultado significa compatibilidad experimental. El error peligroso es tratar resistencias comerciales como valores exactos. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 5. Datos y graficas

Aparece en pendientes ajustadas, barras de error y dispersion. Se calcula que variacion pertenece al ruido y cual a una tendencia. El resultado orienta conclusiones. El error peligroso es forzar una conclusion con datos cuya incertidumbre no la sostiene. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.
