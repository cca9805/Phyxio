# Practice

## Level 1: recognition

Identify in three formulas where an exponential describes change by factors, and a logarithm measures how many factors separate two quantities. Write which quantity depends on which and which condition remains fixed.

Exercise: take one formula from kinematics, one from energy, and one from fluids. Mark the unknown, data, and expected unit before calculating.

## Level 2: direct application

Apply the tool in a case where all data are coherent. Calculate slowly and preserve units at each step.

Exercise: use a simple numerical example from the topic and check that the result has the expected unit or scale.

## Level 3: choosing the tool

Choose between solving, comparing, estimating, checking dimensions, or solving a system. Explain why the chosen tool answers better than the others.

Exercise: ask two questions about the same formula: one requiring direct calculation and another requiring scale comparison.

## Level 4: physical interpretation

Interpret what happens if an input increases, decreases, or changes sign. Link that variation to a concrete physical phenomenon.

Exercise: change one datum by a factor of two and explain whether the result should double, decrease, remain fixed, or change in another way.

## Level 5: intentional error

Analyse a result with a reasonable number but an incorrect unit. Detect the step where interpretation broke, correct the operation, and explain why the corrected result answers the question.

Exercise: invent a plausible error for this topic, write why it seems reasonable, correct the calculation, and state which false physical consequence it would have produced.

## Deepening practice

Basic level: invent a simple numerical example related to logarithms and exponentials, solve it, and write one sentence with unit and meaning.

Intermediate level: change one datum by a factor `10` and predict the new result before calculating. Then verify whether the prediction matches.

Advanced level: write a limiting case where the usual procedure may fail. Explain which physical or mathematical condition is no longer satisfied.

## Intensive practice: factors, times, and slopes

Basic level: a quantity is multiplied by `1/2` every interval. Calculate the fraction remaining after 1, 2, 3, and 4 intervals and explain why this is not constant subtraction.

Intermediate level: solve `N = N0 e^(-lambda t)` for `t` and calculate the time needed to reach `N/N0 = 0.10` if `lambda = 0.20 s^-1`.

Advanced level: a plot of `ln(A/A0)` against `t` has slope `-0.50 s^-1`. Interpret the unit of the slope and calculate the characteristic time.

Very advanced level: compare an exponential model with a power law. Explain which graph you would use to distinguish them with experimental data: semilog or log-log.
