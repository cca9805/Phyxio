# Physical applications

## 1. Gravitation

It appears in the inverse-square law. It calculates how gravitational force changes when the distance between masses changes. The result means relative intensity, not a new force without a model. The dangerous error is treating a 1/r^2 relation as if it were 1/r. As a final control, the learner must explain which datum changes, which quantity is calculated, and which physical conclusion can accept or reject the result.

## 2. Electric fields

It appears when comparing fields produced by point charges. It calculates how much the field decreases with distance. The result gives the spatial scale of influence.
