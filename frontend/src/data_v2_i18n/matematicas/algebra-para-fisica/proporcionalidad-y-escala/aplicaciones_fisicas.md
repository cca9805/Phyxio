# Aplicaciones fisicas

## 1. Gravitacion

Aparece en la ley inversa cuadratica. Permite calcular como cambia la fuerza gravitatoria al cambiar la distancia entre masas. El resultado significa intensidad relativa, no una fuerza nueva sin modelo. El error peligroso es tratar una relacion 1/r^2 como si fuera 1/r. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 2. Campos electricos

Aparece al comparar campos producidos por cargas puntuales. Se calcula cuanto disminuye el campo al alejarse. El resultado indica escala de influencia espacial. El error matematico peligroso es olvidar que la distancia entra elevada al cuadrado. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 3. Ondas

Aparece en v = f lambda y en intensidades que se reparten sobre superficies. Se calcula como frecuencia, longitud de onda e intensidad cambian entre si. El resultado permite interpretar medios y fuentes. El error peligroso es mezclar proporcionalidad directa con inversa. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 4. Fluidos

Aparece en caudales, presiones y escalas geometricas. Se calcula como cambian areas, volumenes y velocidades al modificar un tamano. El resultado significa efecto de escala. El error peligroso es creer que area y volumen crecen igual que la longitud. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 5. Energia y potencia

Aparece al comparar energia transferida, tiempo y potencia. Se calcula si una maquina entrega el doble de energia, tarda la mitad o necesita mayor potencia. El resultado aclara dependencia operativa. El error peligroso es interpretar toda relacion como lineal. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.
