# Practice

## Level 1: recognition

Identify the type of relation before calculating. In `F = m a`, if `a` is fixed, `F` is directly proportional to `m`. In `p = F/A`, if `F` is fixed, `p` is inversely proportional to `A`. Write which quantity changes, which quantity responds, and which condition is held fixed.

Exercise: classify these relations as direct, inverse, quadratic, or cubic: `E = P t`, `I = P/A`, `F = G m_1 m_2/r^2`, `V = L^3`. Do not substitute numbers yet; only read the structure.

## Level 2: direct application

Apply scale factors. If mass doubles while acceleration stays fixed, force doubles. If area doubles while force stays fixed, pressure is halved. Always keep the fixed condition visible because without it the conclusion can change.

Exercise: a wave satisfies `v = f lambda` in the same medium. If frequency triples, calculate by scaling what happens to wavelength. Then explain why the length unit does not change even though the value is divided by three.

## Level 3: choosing the tool

Choose between calculating an absolute value and comparing by scale. If the problem asks "what is the force", you need data and a formula. If it asks "what happens when distance doubles", a proportionality ratio is often enough.

Exercise: for `F = G m_1 m_2/r^2`, decide whether to rearrange, substitute, or compare by factor in these questions: exact force between two masses, effect of doubling `m_1`, effect of tripling `r`, and comparison between two configurations.

## Level 4: physical interpretation

Interpret factors as physical behavior. An inverse-square law does not merely say that something decreases; it says that moving twice as far reduces the effect to one quarter. A cubic relation does not merely say that something grows; it says that doubling a length multiplies volume by eight.

Exercise: a geometric model has all lengths increased by factor `2`. Calculate how length, area, volume, and mass change if density remains fixed. Explain why this result matters in biomechanics, structures, or fluids.

## Level 5: intentional error

Analyze the error: someone says that if distance from a source doubles, any effect also doubles. Detect that distance change has been confused with direct proportionality. Correct the reasoning for a `1/r` law and for a `1/r^2` law.

Analyze another error: someone scales a model by doubling lengths and claims that mass, area, and volume also double. Correct the relation: area scales with the square and volume with the cube. Explain which false physical consequence that error would produce.

## Mastery path

Basic level: you should recognize whether a relation increases, decreases, or stays constant when one variable changes. The first milestone is distinguishing "larger" from "proportional": not every increase is linear and not every decrease is simple inverse behavior.

Intermediate level: you should identify direct, inverse, quadratic, cubic, and power-law proportionality in formulas and graphs. At this level you can predict what happens when a variable doubles, halves, or changes by a factor of ten.

Advanced level: you should use scaling to compare models, detect dominant variables, and decide which term controls a physical regime. An advanced conclusion always states the scale factor, the interval where it applies, and which parameter is being held fixed.

## Deepening practice

Basic level: invent a simple numerical example related to proportionality and scaling, solve it, and write one sentence with unit and meaning.

Intermediate level: change one datum by a factor `10` and predict the new result before calculating. Then verify whether the prediction matches.

Advanced level: write a limiting case where the usual procedure may fail. Explain which physical or mathematical condition is no longer satisfied.
