# Physical applications

## 1. Circuits

It appears when combining Kirchhoff laws with resistance relations. It calculates unknown currents and voltages. The result means an electrical state compatible with all nodes and loops. The dangerous error is writing dependent equations and believing there is enough information. As a final control, the learner must explain which datum changes, which quantity is calculated, and which physical conclusion can accept or reject the result.

## 2. Static equilibrium

It appears when a body has several forces and torques. It calculates reactions, tensions, or support forces. The result means simultaneous translational and rotational equilibrium. The dangerous error is using only force balance and forgetting moments. As a final control, the learner must explain which datum changes, which quantity is calculated, and which physical conclusion can accept or reject the result.

## 3. Collisions

It appears when several quantities are conserved or change together. It calculates coupled final velocities. The result interprets transfer of motion and energy. The dangerous error is mixing an inelastic collision with elastic energy equations. ## 4. Fluids

It appears when combining continuity, Bernoulli, and boundary conditions. It calculates speeds, pressures, or heights. The result means compatibility between flow rate and energy per unit volume. ## 5. Constrained dynamics

It appears when accelerations, tensions, and forces are connected. It calculates unknowns that depend on each other. The result shows the joint response of the system.
