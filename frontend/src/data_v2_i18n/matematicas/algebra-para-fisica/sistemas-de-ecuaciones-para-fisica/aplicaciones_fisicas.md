# Aplicaciones fisicas

## 1. Circuitos

Aparece al combinar leyes de Kirchhoff con relaciones de resistencias. Se calculan corrientes y tensiones desconocidas. El resultado significa un estado electrico compatible con todos los nodos y mallas. El error peligroso es escribir ecuaciones dependientes y creer que hay informacion suficiente. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 2. Equilibrio estatico

Aparece cuando un cuerpo tiene varias fuerzas y torques. Se calculan reacciones, tensiones o fuerzas de apoyo. El resultado significa equilibrio simultaneo traslacional y rotacional. El error peligroso es usar solo suma de fuerzas y olvidar momentos. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 3. Colisiones

Aparece cuando varias magnitudes se conservan o cambian juntas. Se calculan velocidades finales acopladas. El resultado interpreta transferencia de movimiento y energia. El error peligroso es mezclar una colision inelastica con ecuaciones de energia elastica. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 4. Fluidos

Aparece al combinar continuidad, Bernoulli y condiciones de contorno. Se calculan velocidades, presiones o alturas. El resultado significa compatibilidad entre caudal y energia por unidad de volumen. El error peligroso es resolver una ecuacion aislada sin la condicion geometrica. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.

## 5. Dinamica con ligaduras

Aparece cuando aceleraciones, tensiones y fuerzas estan conectadas. Se calculan incognitas que dependen unas de otras. El resultado muestra la respuesta conjunta del sistema. El error peligroso es tratar cada cuerpo como si estuviera aislado. Como control final, el alumno debe explicar que dato cambia, que magnitud se calcula y que conclusion fisica permite aceptar o rechazar el resultado.
