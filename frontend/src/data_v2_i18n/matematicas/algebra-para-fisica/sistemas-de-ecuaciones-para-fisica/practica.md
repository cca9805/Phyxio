# Practica

## Nivel 1: reconocimiento

Identifica en tres formulas donde aparece un sistema aparece cuando una sola ecuacion no contiene toda la informacion necesaria. Escribe que magnitud depende de cual y que condicion se mantiene fija.

Ejercicio: toma una formula de cinematica, una de energia y una de fluidos. Marca variable buscada, datos y unidad esperada antes de calcular.

## Nivel 2: aplicacion directa

Aplica la herramienta en un caso donde todos los datos sean coherentes. Calcula despacio y conserva unidades en cada paso.

Ejercicio: usa un ejemplo numerico sencillo del tema y comprueba que el resultado tiene la unidad o escala esperada.

## Nivel 3: eleccion de herramienta

Elige entre despejar, comparar, estimar, comprobar dimensiones o resolver un sistema. Explica por que la herramienta elegida responde mejor que las otras.

Ejercicio: plantea dos preguntas sobre la misma formula: una que pida calculo directo y otra que pida comparacion de escala.

## Nivel 4: interpretacion fisica

Interpreta que ocurre si una entrada aumenta, disminuye o cambia de signo. Relaciona esa variacion con un fenomeno fisico concreto.

Ejercicio: cambia un dato por un factor dos y explica si el resultado debe duplicarse, reducirse, permanecer igual o cambiar de otra forma.

## Nivel 5: error intencionado

Analiza un resultado que tiene numero razonable pero unidad incorrecta. Detecta el paso donde se rompio la interpretacion, corrige la operacion y explica por que el resultado corregido si responde la pregunta.

Ejercicio: inventa un error plausible para este tema, escribe por que parece razonable, corrige el calculo y senala que consecuencia fisica falsa habria producido.

## Practica de profundizacion

Nivel basico: inventa un ejemplo numerico sencillo relacionado con sistemas de ecuaciones para fisica, resuelvelo y escribe una frase con unidad y significado.

Nivel intermedio: cambia un dato por un factor `10` y predice el nuevo resultado antes de calcularlo. Luego verifica si la prediccion coincide.

Nivel avanzado: escribe un caso limite donde el procedimiento habitual pueda fallar. Explica que condicion fisica o matematica deja de cumplirse.

## Practica intensiva: de una ecuacion aislada a un sistema

Nivel basico: escribe dos ecuaciones para dos masas unidas por una cuerda ideal. Identifica que magnitud comparten y que fuerza interna aparece con signos opuestos.

Nivel intermedio: plantea un encuentro entre dos moviles con `x1 = x10 + v1 t` y `x2 = x20 + v2 t`. Resuelve el tiempo imponiendo `x1 = x2`.

Nivel avanzado: plantea un sistema energia-momento para una colision unidimensional elastica. No hace falta resolverlo completo: identifica incognitas, ecuaciones y restricciones.

Nivel muy avanzado: explica cuando un sistema lineal deja de tener solucion unica. Relaciona determinante cero con ecuaciones fisicamente dependientes o datos incompatibles.
