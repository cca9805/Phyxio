# Procedure

## Before calculating

Read the statement and decide which quantity is requested. Then separate data, constants, and conditions that remain fixed. This decision matters because a system appears when one equation does not contain all the necessary information only makes sense when you know what is changing.

Also write the expected units. If the result must be a force, energy, time, or dimensionless ratio, that information controls the whole procedure.

## Step 1: identify the mathematical object

Look for the dominant structure: product, quotient, power, sum, system, scale, or factor. Do not start by substituting. First identify which mathematical object organises the physical information.

If several variables appear, mark the unknown and the parameters. In physics, confusing a parameter with a variable often changes the conclusion.

## Step 2: choose the operation

Choose the operation that answers the question: compare, solve, estimate, check dimensions, solve several equations, or apply a factor. Not every question asks for the same calculation.

The correct tool must preserve the physical relation. If one quantity changes and the question asks for the effect on another, the operation must reflect that dependence.

## Step 3: preserve units and signs

Carry units throughout the calculation. Units are not decoration: they show whether a sum, quotient, power, or logarithm makes sense. If an operation produces an impossible unit, the error has already appeared.

Also preserve signs when they represent direction, loss, sense, or difference. Taking absolute values without justification can hide physical information.

## Step 4: calculate

Calculate in an ordered way and make the transformation step visible. If there are scale factors, powers, or systems, avoid jumping directly to the numerical result.

When possible, estimate first. A prior estimate gives the expected scale and helps detect calculator, conversion, or rounding errors.

## Step 5: interpret the result

Write a complete physical sentence. The result must state which quantity was obtained, with which unit, under which condition, and whether its scale is reasonable.

Then check the dependence: if an input increases, the result should move in the direction predicted by the model. This reading turns calculation into understanding.

## Self-check checklist

Check that you identified the unknown, data, and constants. Verify that the chosen operation matches the question. Review units, sign, domain, and scale.

Before finishing, explain what a larger, smaller, or opposite-sign result would mean. If that explanation is not clear, return to the model before trusting the number.

## Expanded didactic route

1. Before calculating, write a verbal prediction: the result should increase, decrease, stay fixed, or change order of magnitude.
2. Separate data, unknown, constants, and constraints. Not every symbol in a formula plays the same physical role.
3. Choose the tool from the structure, not from habit. In systems of equations for physics, the form of the relation decides the procedure.
4. Calculate while keeping units, powers, and signs visible. If one line hides too much, split the step.
5. Interpret the result in one sentence: what it measures, in what unit, under what condition, and at what scale.

To close the exercise, mentally change one datum by a simple factor and check whether the result moves as the model predicts.
