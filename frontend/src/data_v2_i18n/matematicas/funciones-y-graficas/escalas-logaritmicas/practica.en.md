# Practice

## Level 1: recognition

Identify on a scale marked `1`, `10`, `100`, `1000` that each step multiplies by `10`. Write how many orders of magnitude separate `0.01` and `1000`.

Classify three graphs as linear, semilog, or log-log according to their axes. Explain which kind of phenomenon you would expect in each one.

## Level 2: direct application

Calculate which intensity factor corresponds to a difference of `20 dB`. Write the equation used and explain why the result is multiplicative.

Calculate how a quantity `y = C x^-2` changes when `x` doubles. Write the reading in words, not only as a number.

## Level 3: choosing the tool

Decide whether you would use linear, semilog, or log-log scale for: radioactive decay, inverse-square law, comparison of astronomical frequencies, and daily temperature. Justify each choice.

Choose whether a table containing zeros can be plotted directly on logarithmic axes. If it cannot, explain which datum, reference, or model must be reviewed.

## Level 4: physical interpretation

Interpret a slope `3` on a log-log graph. Explain what happens to `y` when `x` is multiplied by `2` and by `10`.

Interpret a pH decrease from `7` to `5`. Calculate the change factor of `H+` concentration and explain the negative sign in the definition.

## Level 5: intentional error

A learner claims that going from `40 dB` to `50 dB` adds the same intensity as going from `80 dB` to `90 dB`. Locate the error and correct the reading as a factor.

Another learner takes logarithms of negative values of an oscillating signal. Explain why this is invalid in real numbers and propose an appropriate positive variable, such as amplitude, power, or RMS value.
