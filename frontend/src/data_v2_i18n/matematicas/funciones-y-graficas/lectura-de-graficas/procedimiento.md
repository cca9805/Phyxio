# Procedimiento

## Antes de calcular

Mira la grafica completa antes de leer puntos. Identifica titulo, ejes, unidades, escala, origen, intervalo mostrado y si los datos son una curva ideal o medidas experimentales. Decide si necesitas un valor puntual, una tendencia, una comparacion, un corte, un maximo o una zona de validez.

Comprueba tambien si el eje es lineal o logaritmico. En fisica, una lectura rapida puede fallar si se ignora que cada division representa una cantidad distinta o si se extrapola fuera del intervalo medido.

## Paso 1: identifica el objeto matematico

Determina que representa cada eje. El eje horizontal suele ser la variable independiente, por ejemplo tiempo, distancia o frecuencia. El eje vertical suele ser la magnitud que responde, como posicion, velocidad, energia, temperatura o amplitud.

Escribe mentalmente la relacion como `y` frente a `x`, pero traduce `x` e `y` a magnitudes fisicas. No digas "sube y"; di "aumenta la velocidad" o "crece la energia almacenada".

## Paso 2: elige la operacion

Si la pregunta pide "cuanto vale", lee una coordenada. Si pide "cuando ocurre", busca el valor horizontal correspondiente. Si pide "como cambia", describe tendencia o pendiente cualitativa. Si pide "donde se anula", busca cortes con el eje horizontal.

No uses pendiente, area o ajuste automaticamente. Esas operaciones pertenecen a otros pasos matematicos. En lectura de graficas, primero decides que informacion visual responde a la pregunta y solo despues eliges si hace falta calcular algo.

## Paso 3: conserva unidades y signos

Cada lectura debe conservar la unidad del eje correspondiente. Un valor tomado del eje vertical usa la unidad vertical; un tiempo tomado del eje horizontal usa la unidad horizontal. Si la grafica cruza valores negativos, el signo tiene significado fisico y no debe convertirse en modulo sin motivo.

Cuando compares dos puntos, revisa que ambos esten en la misma escala. Si un eje usa divisiones de 0.2 y otro de 5, contar cuadros sin leer etiquetas produce errores grandes.

## Paso 4: calcula

Para leer un punto, traza una linea vertical desde el valor horizontal hasta la curva y luego una horizontal hacia el eje vertical. Si el punto cae entre marcas, interpola de forma razonable y declara que el resultado es aproximado.

Para reconocer una tendencia, divide la grafica en tramos: creciente, decreciente, constante, curvado hacia arriba, curvado hacia abajo u oscilante. Para cortes y extremos, localiza el punto y lee sus coordenadas con unidad.

## Paso 5: interpreta el resultado

Convierte la lectura en una frase fisica. Por ejemplo: "a los 3 s, el movil esta cerca de 6 m", "la temperatura se estabiliza despues de 80 s" o "la amplitud maxima aparece al aumentar la frecuencia hasta la resonancia".

Indica si la lectura es exacta, aproximada o dependiente del modelo. Si la grafica procede de datos experimentales, menciona dispersion, incertidumbre visual o tramo donde el patron parece fiable.

## Checklist de autocontrol

- Identifica que magnitud y unidad tiene cada eje.
- Comprueba si la escala es lineal, logaritmica, regular o irregular.
- Distingue valor puntual, tendencia, corte, extremo y tramo.
- Conserva signo y unidad en cada lectura.
- Evita extrapolar fuera del dominio mostrado.
- Explica el resultado como una frase fisica, no como una coordenada aislada.
