# Physical applications

## 1. Springs

It appears when fitting force versus extension in the elastic region of a spring. The slope `k` is calculated and means spring constant. The dangerous error is including large deformations where the spring no longer obeys a linear law and concluding a false constant.

## 2. Gases

It appears when plotting pressure versus temperature or inverse volume. A proportionality constant is calculated, connecting the gas to the ideal model. The result means behavior compatible with a simple law. The dangerous error is mixing Celsius temperatures with absolute scales when the model requires kelvin.

## 3. Decay and cooling

It appears when plotting `ln(y)` versus time. A decay rate or time constant is calculated. The result means loss proportional to the current state. The dangerous error is fitting a line to original data and turning a relative rate into a false absolute rate.

## 4. Fields and intensity

It appears in power laws, such as intensity versus distance. An exponent is calculated using a log-log graph. The result means how the quantity changes when distance is scaled. The dangerous error is ignoring offsets or experimental background before taking logarithms.

## 5. Instrument calibration

It appears when comparing a sensor reading with known values. Calibration slope and intercept are calculated. The result converts readings into real quantities. The dangerous error is deleting the intercept because it is inconvenient, when it may represent systematic offset.
