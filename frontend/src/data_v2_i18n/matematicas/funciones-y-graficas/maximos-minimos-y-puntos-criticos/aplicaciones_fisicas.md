# Aplicaciones fisicas

## 1. Energia potencial

Aparece al estudiar equilibrio. Se calculan minimos y maximos de energia. El resultado significa estabilidad si el minimo aumenta hacia ambos lados, o inestabilidad si el punto es maximo. El error peligroso es llamar estable a cualquier punto donde la pendiente sea cero.

## 2. Movimiento de proyectiles

Aparece al buscar altura maxima o alcance optimo. Se calcula el extremo de una trayectoria o de una expresion dependiente del angulo. El resultado significa condicion de valor maximo dentro del modelo. El error peligroso es olvidar resistencia del aire o dominio de angulos permitidos.

## 3. Resonancia

Aparece en graficas de amplitud frente a frecuencia. Se calcula el maximo de respuesta y se identifica la frecuencia de resonancia dentro del intervalo excitado. El resultado significa frecuencia donde el sistema absorbe energia con mayor eficacia. El error peligroso es confundir maximo de amplitud con estabilidad del sistema.

## 4. Termodinamica y estabilidad

Aparece en potenciales termodinamicos. Se calculan minimos compatibles con restricciones como temperatura, presion o volumen fijados. El resultado significa estado estable o equilibrio bajo condiciones fijadas. El error peligroso es ignorar que cambia la funcion relevante segun variables controladas.

## 5. Optimizacion experimental

Aparece al buscar maxima potencia, minima perdida o mejor rendimiento. Se calcula una condicion optima dentro de datos medidos. El resultado guia diseño y decision. El error peligroso es aceptar un extremo aparente producido por ruido o por pocos puntos.
