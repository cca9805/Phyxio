# Practica

## Nivel 1: reconocimiento

Identifica en tres curvas los maximos, minimos y puntos donde la curva se aplana. Escribe si cada extremo parece cima, valle o punto sin extremo.

Marca en una grafica de energia potencial que punto seria equilibrio estable y cual seria inestable. Justifica usando la forma alrededor del punto.

## Nivel 2: aplicacion directa

Para `h(t) = 12t - 3t^2`, calcula el instante de altura maxima y la altura maxima. Escribe unidades en ambos resultados.

En una tabla de potencia frente a resistencia, localiza el valor maximo medido. Indica si puedes llamarlo maximo global o solo maximo dentro de la tabla.

## Nivel 3: eleccion de herramienta

Decide si basta leer la grafica o si necesitas derivar para: altura maxima de un proyectil, minimo de energia potencial, frecuencia de resonancia y mejor ajuste experimental. Justifica cada decision.

Compara un maximo local y un maximo global en una curva con dos cimas. Explica cual responde una pregunta de optimizacion completa.

## Nivel 4: interpretacion fisica

Interpreta un minimo de energia en `x = 0`. Explica que ocurriria si el sistema se desplaza ligeramente y por que eso se llama estabilidad.

Interpreta un maximo de amplitud en una curva de resonancia. Explica que magnitud se maximiza y que riesgos fisicos podria implicar.

## Nivel 5: error intencionado

Un alumno encuentra `f'(x) = 0` y concluye automaticamente que hay un minimo estable. Localiza el error, clasifica el punto usando signos de pendiente o curvatura y corrige la conclusion.

Otro alumno busca un maximo fuera del intervalo de datos medidos y lo usa para diseñar un experimento. Explica el problema de dominio y redacta una recomendacion limitada al intervalo observado.
