# Aplicaciones fisicas

## 1. Cinematica

Contexto fisico: movimiento con posicion, velocidad o trayectoria. Uso operativo: calcula componentes de desplazamiento o velocidad, estima la escala y compara si el signo coincide con el eje elegido. En este contexto, triangulos y relaciones geometricas conecta el dibujo con una magnitud medible y evita usar una formula sin geometria.

La transferencia importante es que unidad, signo, escala y modelo deben sobrevivir al calculo. Si la componente, distancia, angulo o razon obtenida contradice el sistema fisico, se revisa la referencia angular, el cuadrante y la relacion elegida antes de confiar en el numero.

## Control final de transferencia fisica

En todas estas aplicaciones, la herramienta aparece para convertir una geometria en una magnitud fisica calculable. El alumno calcula una componente, distancia, angulo o razon, pero despues interpreta el resultado con unidad, signo, escala y modelo. La interpretacion debe decir que significa el numero dentro del sistema fisico y que error evita: usar mal el cuadrante, elegir una funcion equivocada, perder una unidad o aceptar un resultado incompatible con el dibujo.
