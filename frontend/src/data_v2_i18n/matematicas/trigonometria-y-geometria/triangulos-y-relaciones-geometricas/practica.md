# Practica

## Nivel 1: reconocimiento

Identifica en un dibujo el angulo, la referencia, el sentido positivo, los catetos, la hipotenusa, el radio o las componentes. Clasifica si el problema pide longitud, componente, angulo, razon o posicion. Explica que unidad esperas antes de calcular.

## Nivel 2: aplicacion directa

Usa la relacion `a^2 + b^2 = c^2, razones iguales en triangulos semejantes` con un dato numerico sencillo. Calcula el resultado, conserva la unidad y compara si la escala es razonable. Repite el ejercicio cambiando el angulo a 30, 45 o 60 grados para usar valores notables.

## Nivel 3: eleccion de herramienta

Elige entre seno, coseno, tangente, Pitagoras, radianes, semejanza o conversion polar en tres situaciones distintas. Justifica la eleccion con una frase geometrica: que representa cada lado, que eje se usa y que dato falta.

## Nivel 4: interpretacion fisica

Interpreta el resultado como componente de fuerza, velocidad, altura, arco, direccion o posicion. Compara que ocurre si el angulo aumenta, si cambia el radio, si se invierte el eje o si se pasa a otro cuadrante. Detecta si el signo tiene sentido.

## Nivel 5: error intencionado

Corrige un planteamiento donde se haya confundido opuesto con adyacente, grados con radianes, signo con modulo o razon trigonometrica con magnitud fisica. Escribe la version corregida y explica que consecuencia fisica tendria el error.

## Practica intensiva de cierre

Nivel basico: calcula una componente sencilla y dibuja el triangulo. Nivel intermedio: estima antes de calcular. Nivel avanzado: usa una funcion inversa y revisa el cuadrante. Nivel muy avanzado: decide si el modelo geometrico sigue siendo valido cuando el angulo o la base cambian con el tiempo.

## Entrenamiento acumulativo

Ejercicio 1: identifica la geometria sin calcular y explica que dato falta. Ejercicio 2: estima el resultado usando un angulo notable. Ejercicio 3: calcula con la relacion simbolica y conserva la unidad. Ejercicio 4: cambia el cuadrante o el eje de referencia y predice el signo. Ejercicio 5: redacta una interpretacion fisica completa. Ejercicio 6: inventa un error plausible y corrigelo justificando la correccion.
