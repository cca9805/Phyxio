# Campos vectoriales basicos

## Que es

Un campo vectorial asigna un vector a cada punto del espacio. No describe una sola flecha, sino una regla espacial: en cada posicion hay modulo, direccion y sentido que pueden cambiar de un punto a otro.

## Por que aparece en fisica

Aparece en gravedad, campo electrico, campo magnetico, velocidad de un fluido y viento. Permite estudiar fuerzas distribuidas, trayectorias posibles, flujo, circulacion y regiones donde el efecto aumenta o disminuye.

## Idea visual o geometrica

La imagen visual es un mapa lleno de flechas. Flechas largas indican mayor modulo, flechas orientadas muestran direccion local y cambios de direccion revelan fuentes, sumideros, remolinos o simetrias.

## Lenguaje simbolico minimo

Se escribe `vec F(x, y, z)` o `vec E(vec r)`. Cada punto de entrada devuelve componentes, por ejemplo `(F_x(x,y), F_y(x,y))`. El campo no es una trayectoria: una trayectoria puede seguir o cruzar el campo segun la dinamica.

## Que significa el resultado

El resultado indica que vector sentiria una particula o un elemento de fluido en cada posicion. Un campo uniforme tiene el mismo vector en todos los puntos; uno radial cambia de direccion con la posicion.

## Cuando no debes usarlo

No debes confundir campo con linea de campo ni con una particula moviendose. Tampoco debes leer flechas dibujadas como valores exactos si solo son representacion cualitativa.

## Resumen operativo

Identifica dominio, vector en cada punto, variacion espacial y simetria. Nivel basico: leer flechas. Intermedio: evaluar componentes en puntos. Avanzado: conectar con flujo, circulacion, divergencia y rotacional.
