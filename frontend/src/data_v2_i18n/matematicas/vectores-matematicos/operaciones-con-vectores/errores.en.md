# Common errors

## Error 1: choosing the wrong operation

It may seem reasonable to use the same calculation for every vector. It fails because addition, projection, and rotation answer different questions. Detect it by reading whether the prompt asks for resultant, component, alignment, or rotation axis. Correct it by choosing the operation before substituting numbers.

## Error 2: losing the meaning of the variable

The error appears when an arrow is treated as a number without reference. A force, velocity, or field has different units and meaning. Detect it by asking which physical quantity each vector represents. Correct it by writing physical name, unit, and coordinate system.

## Error 3: ignoring units

Vector operations preserve or combine units. A force component remains in newtons, work comes out in joules, and torque in newton meters. Detect it by calculating final units. Correct it by operating with units together with numbers.

## Error 4: confusing sign and magnitude

A magnitude cannot be negative, but a component can. Deleting signs destroys direction and sense. Detect it by comparing the sign with the drawing. Correct it by separating magnitude, component, and physical sense.

## Error 5: applying a rule outside its domain

Some rules are valid only in orthogonal bases, with vectors in the same system, or in three dimensions. Detect it by checking assumptions about axes, plane, and domain. Correct it by declaring basis, dimension, and limits before operating.

## Quick detection rule

If a vector answer does not state direction, sense, unit, and meaning, it is incomplete. The minimum control is drawing the result and checking whether it matches the expected physics.
