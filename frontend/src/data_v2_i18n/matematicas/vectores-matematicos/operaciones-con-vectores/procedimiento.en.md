# Procedure

## Before calculating

Read the physical situation before operating. Identify whether you need magnitude, direction, component, addition, projection, rotation, change of basis, or field. With vectors, choosing the wrong operation changes the meaning even if the number looks reasonable.

## Step 1: identify the mathematical object

Write which vector or vectors appear, which units they have, and relative to which reference they are described. Draw an arrow if geometry is involved. If there are components, declare axes and positive senses before calculating.

## Step 2: choose the operation

Choose the tool from the question: addition for resultant, subtraction for comparison, scalar multiplication for magnitude change, dot product for alignment, cross product for rotation, change of basis for new axes, and field for a vector at each point.

## Step 3: preserve units and signs

Preserve units in components and results. Magnitude has no sign, but components do. If you change basis, do not mix old and new components. If a product appears, check whether the final unit is a product or quotient.

## Step 4: calculate

Calculate step by step. With components, operate axis by axis. With products, decide first whether cosine or sine is needed. With fields, evaluate the point before interpreting the arrow. Check the result with an approximate drawing.

## Step 5: interpret the result

Turn the result into a physical sentence: which direction it has, which sense it indicates, which unit it preserves, and which effect it produces. If the result is zero, explain cancellation, perpendicularity, equilibrium, or absence of component.

## Self-check checklist

Axes declared, units visible, operation justified, signs checked, drawing coherent, result interpreted, and domain stated. Basic level: arrows. Intermediate level: components. Advanced level: bases, products, and fields with physical meaning.
