# Aplicaciones fisicas

## 1. Movimiento en 2D

Aparece cuando una magnitud dirigida debe calcularse, compararse o interpretarse en movimiento en 2d. Se calcula una componente, resultante, proyeccion, orientacion o vector asociado segun la pregunta. El resultado significa un efecto fisico con modulo, direccion, sentido y unidad. El error peligroso es reducirlo a un numero sin conservar la geometria vectorial.
