# Phyxio (migracion V2)

Esta carpeta contiene una migracion operativa para validar V2 antes de limpiar el resto del repo.

## Estructura
- backend/
- frontend/

## Arranque local
1. Backend
   - cd backend
   - npm install
   - npm run dev
2. Frontend
   - cd frontend
   - npm install
   - npm run dev

## Variables de entorno
- Backend: usar backend/.env.example como base para backend/.env
- Frontend: revisar frontend/.env (VITE_API_URL)
