# Aplicaciones
## 1. Enfriamiento de objetos cotidianos
Variable dominante: [[variacion_entropia_universo]]
Límite de validez: [[variacion_entropia_universo]] >= 0
Cuando una taza caliente se enfría, se mide temperatura, pero se interpreta dirección temporal mediante entropía total. La taza puede perder energía interna, mientras el aire gana entropía. La aplicación predice qué dirección ocurre espontáneamente y restringe interpretaciones que miran solo el objeto visible.

## 2. Difusión y mezcla
Variable dominante: [[numero_microestados]]
Límite de validez: [[numero_microestados]] > 1

Se mide concentración, pero se explica la dirección con [[probabilidad_macroestado]] y [[entropia]]. La mezcla inversa no es imposible por lógica, sino físicamente inobservable por probabilidad.

## 3. Fricción mecánica
Variable dominante: [[entropia_producida]]
Límite de validez: [[entropia_producida]] > 0
En un bloque que se desliza y se detiene, la energía mecánica se dispersa térmicamente. La magnitud relevante es [[entropia_producida]], que cuantifica el alejamiento del límite reversible. La aplicación predice que el movimiento no se recupera espontáneamente sin cambiar el entorno.

## 4. Máquinas térmicas reales
Variable dominante: [[grado_irreversibilidad]]
Límite de validez: [[grado_irreversibilidad]] >= 0
Una máquina real nunca opera con reversibilidad perfecta. Rozamientos, diferencias finitas de temperatura y pérdidas producen entropía. El [[grado_irreversibilidad]] permite comparar diseños y detectar cuánto se alejan del ideal. Se constriñe la eficiencia posible mediante la segunda ley.

## 5. Fluctuaciones microscópicas
Variable dominante: [[constante_boltzmann]]
Límite de validez: escala comparable con múltiplos de [[constante_boltzmann]]
En sistemas pequeños pueden aparecer fluctuaciones que parecen invertir la tendencia local. La aplicación exige comparar los cambios con [[constante_boltzmann]]. En macroescala, esas fluctuaciones no invalidan la flecha temporal; solo muestran que la segunda ley tiene raíz estadística.
