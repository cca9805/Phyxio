# Probability and Ensembles

## Conceptual errors

### Error 1: Treating the ensemble as one real object

**Why it seems correct**
It seems plausible because the [[ensemble]] is often described as many copies and the drawing looks like a physical collection.

**Why it is incorrect**
The [[ensemble]] is a statistical construction. It organizes [[probabilidad_estado]] over compatible [[estado_microscopico]], but it is not one material object.

**Detection signal**
The solution treats [[numero_copias_ensemble]] as a mandatory literal number or uses [[conteo_sistemas]] without checking normalization.

**Conceptual correction**
Start from the [[restriccion_macroscopica]], define compatible states, then assign probabilities, compute [[valor_medio_ensemble]], and read [[fluctuacion_ensemble]].

**Mini-example of contrast**
If two states have weights 1 and 3, saying there are two states is not enough. The larger [[peso_estadistico]] contributes more to the [[valor_medio_ensemble]].
